노뮤트 — 하단 네비게이션 시안

네가 준 레퍼런스에 노뮤트 UIUX(다크 글래스·#0FFD02·Pretendard)만 입힌 4안. 활성탭=메인. 아이콘은 이모지 말고 라인 아이콘으로 격상.
맘에 드는 번호만 말해줘 — 그대로 실뷰어에 박는다. 조합·디테일 수정도 OK.

1안 글래스 알약 활성
활성 탭을 은은한 초록 글래스 알약으로 감쌈. 차분·고급.
메인 스크랩 통계 설정
2안 상단 인디케이터 바
활성 탭 위 짧은 네온 바. 깔끔·미니멀, 위치 인지 빠름.
메인 스크랩 통계 설정
3안 아이콘 부양 + 글로우
활성 아이콘이 살짝 떠오르고 네온 글로우 + 하단 도트. 생동감.
메인 스크랩 통계 설정
4안 중앙 FAB(슛/＋)
가운데 떠오른 네온 버튼 = 빠른 동작(예: 슛·새 글). 좌2·우2 탭.
메인 스크랩 통계 설정
고른 뒤 → 실뷰어 하단에 그대로 적용 + 탭 전환(메인↔스크랩 등) 배선. 2번 스크랩 탭은 "준비중" 플레이스홀더로 먼저 깔아둘게(스크래핑 DB 자리). 번호 + "이건 이렇게" 디테일 있으면 같이 말해줘.
