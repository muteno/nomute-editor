# 새로운 자막 편집기 소개

## ⓪ 개요
- **내용**: 새 자막 편집기 소개 영상 — 영상에서 음성을 추출해 자동으로 자막을 만들고, 단어별 타이밍·탭 편집·찾아 바꾸기·조각 합치기/나누기·SRT 내보내기로 릴스/쇼츠 제작을 빠르게 해준다는 홍보.
- **화자**: 1인(소개자·내레이터) — 단일 화자라 색점 없음.
- **메타**: 0:55 · 한국어 · 한국어 원본 모드 · 조각 29개
- **⚠ STT 교정**: 입력에 음성인식 오류가 많아 문맥으로 복원함(추구추구→추출, 리우스·수우→릴스·쇼츠, 캡커신아→캡컷이나, 팔아→빨라 등). 확신 낮아 확인 필요한 2곳 — ①(3번) `편집기`(원음 `변기피` → 제품명일 가능성) ②(15번) `나누는`(원음 `치우는` → `삭제/빼기`일 수도).

「원본 구절 수: 10개 → 자막 조각 수: 29개」 (긴 문장 호흡 분할) 「⚠ 조각 타임코드 추정」
