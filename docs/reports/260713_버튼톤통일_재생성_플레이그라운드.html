버튼 톤 통일 + 재생성

레포에 흩어진 버튼들(생성·수정·발행·전송 + 재생성 아이콘)을 한 톤으로. 아래 방식 골라 — 맨 밑 전송(게이지)은 발사 중 상태라 초록 대비 확인용.

텍스트 CTA (모달 발사류)

이미지 위 액션

전송 ← 발사 중(초록 게이지) 대비 확인

톤 통일 방식

전부 정렬 근본(방금 반영)은 공통. 배경·테두리 톤만.

  • ⚠️ 게이지 대비: 지금 .asksend가 검정 글래스인 건 발사 중 초록 게이지가 또렷하게 차오르라고 그런 거야(맨 밑 전송 버튼). 흰 글래스로 바꾸면 대비가 약해져 — 그래서 B(흰글래스)냐 C(검정 유지)냐가 핵심 갈림.
  • 재생성·저장 아이콘 버튼은 이미지 위라 불투명 플레이트가 기본(뒤 사진에 안 묻힘) — 톤만 테두리·글자색으로 맞춰.
  • 고른 방식 선택값 복사 → 레포 전반 해당 버튼 일괄 통일할게(꽤 많으니 한 번에).
  • 색 = viewer :root 실측 · 폰트 시스템 폴백.
