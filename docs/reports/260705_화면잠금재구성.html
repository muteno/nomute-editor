화면잠금 재구성 + 잠금화면 오션 배경

260705 · viewer/index.html · check_refs.py · PR #1674 (머지 4147434) · §검증 10인 평의회
잠금화면 = 오션(파랑·시안) 물결 위 PIN — 물결은 천천히 흐름

무엇을 바꿨나

잠금을 프로필 메뉴에서 바로 켜고 끄는 ON/OFF 토글로 바꿨어(알림처럼). 켤 때 PIN을 바로 만들고, 끌 때 기존 PIN을 확인해. 세부 설정 팝업에선 ON/OFF를 없애고 자동잠금 게이지 + PIN 변경만 남겼어. 잠금화면 배경은 아까 그 블루(오션) 물결로 깔았고, 잠금이 걸린 기기는 부팅 때 인트로 없이 바로 PIN이 떠. 덤으로 부팅 스플래시의 프로필 얼굴을 빼 깔끔하게 하고, 곳곳에 raw로 박혀 있던 SNS 시안색을 팔레트 토큰으로 통일했어.

전 / 후

항목전후
잠금 ON/OFF팝업(잠금 설정 창) 안에 숨어 있음프로필 메뉴에 토글로 노출 · OFF→ON=즉시 PIN 설정 / ON→OFF=PIN 확인
세부 팝업ON/OFF + 게이지 + PIN 설정/해제ON일 때만 진입 · 게이지 + PIN 변경만(ON/OFF 제거)
잠금화면 배경챠콜 90% + 블러(정적 다크)오션 물결(파랑·시안) + 옅은 블루-딤
부팅(잠금 시)인트로 스플래시 → 이후 PIN인트로 스킵 → 바로 PIN(첫 페인트 전 판정=플래시 0)
부팅 스플래시프로필 얼굴 + 앱이름앱이름만 중앙(얼굴 제거)
SNS 시안색raw #0cd0f7·rgba(12,208,247) 곳곳var(--naver)·var(--naver-rgb) 토큰(픽셀 동일)

효율성 · 성능

  • 물결 재활용(추가 비용 0) — 잠금화면은 이미 도는 배경 셰이더를 z-index만 올려 앞으로 가져옴(캔버스 재초기화·컨텍스트 로스트 없음 = 순수 리컴포짓)
  • 잠금 중 30fps 다운시프트 — 유휴 잠금은 사용자가 자리를 뜬 상태 → 평소 120fps를 잠금화면에선 30fps로 낮춰 배터리 방어(평의회7 지적)
  • 픽셀 회귀 0 토큰화 — SNS 시안 raw→토큰은 값이 동일(#0cd0f7=rgb(12,208,247)) → 화면 무변화, 게이트 경고만 해소
  • check_refs baseline 래칫 — blur 134→132(잠금 blur 제거), hex 163→157(시안 토큰화)

10인 평의회 발견 · 수정

회귀 수정 토글을 팝업 밖으로 옮기면서 설정 메뉴 열 때 갱신 트리거가 빠져 토글이 실제 ON인데 "OFF"로 표시되던 회귀 → 메뉴 펼침에 lkSettingsRefresh() 추가(실클릭 흐름 headless 확인)
회귀 수정 '잠금 설정' 라벨이 전폭 버튼→작은 텍스트로 탭 타깃 축소 → flex:1로 히트영역을 행 폭까지 복원(시각 무변화)
복구경로 마스터 PIN(898900)이 5회잠금 중에도 통하게(잠금화면과 통일)
모바일 PIN 입력 중 프로필 메뉴 유지 → 설정 후 토글 갱신이 보임
견고성 잠금화면을 .topbanner 밖 body 직속으로 이동(조상 스태킹 컨텍스트 함정 제거)
레이스 잠금 중 모달 진입 차단 · 토글 재진입 가드 · ON→OFF async 잠김 방어
가독 잠금화면 문구 text-shadow + 스크림 소폭 상향(움직이는 밝은 물결 위 대비 확보)
접근성 PIN 오류에 role="alert"(스크린리더 낭독)

품질저하 리스크

  • 주의 잠금화면 가독성 트레이드오프 — 정적 다크(90%+블러)에서 투명 오션 물결로 바뀌어 대비 하한을 스크림(.34)+text-shadow로 확보. 실기기에서 흐리면 스크림 상향 여지.
  • 주의 마스터 PIN이 이제 설정창에서도 5회잠금을 우회 — 잠금화면과 동일하게 맞춘 것(복구 일관성). 로컬 가림막 성격상 의도된 방향.
  • 주의 부팅 인트로 스킵 판정은 스플래시 표시만 담당(실제 잠금은 메인 스크립트가 소유) — 어긋나도 보안 영향 0(앱은 booted로 정상 노출).
  • 데드락·언락불가·프라이버시 약화·z가림 실패·메모리 누수 = 없음(평의회 실측). OFF는 PIN을 지우지만 인증 뒤에만 실행돼 새 우회 없음.
