한계·전제 각주
  • 엔진: Real-ESRGAN(CPU) = 저화질 이미지 스위트스팟. 큰 이미지는 auto 가드가 FSRCNN/Lanczos로 자동 우회(느림 방지) — 이미지 스튜디오 소스는 대개 저화질이라 딱 맞음.
  • 배율: 4배 = Real-ESRGAN 네이티브. 2·3배 = 4배 후 축소(품질 그대로, 파일만 작게).
  • 발사 백엔드(선택 후 배선): 신규 api/upscale.js + img-upscale.yml = 기존 리사이즈 3층 미러. 워크플로에 FX_ESRGAN=1로 sha256 핀 모델 드롭. 잡·폴링·이력 = 리사이즈/모자이크 계승.
  • 이 페이지는 배치 시안만 — 발사·잡·폴링은 자리 확정 후 붙인다. 점선 강조는 시안 표식(실제 UI엔 없음).
