전체 다운로드 버튼 — 첨부1 · 첨부2 · 통합안(첨부3)

같은 카드뉴스 요약 안의 두 '전체' 버튼이 디자인·완료동작이 서로 달라. 각 버튼을 탭하면 실제 동작(순차 저장 → 완료)이 재생돼. 통합 판단용.

첨부1 카드 피드 섹션 「전체」 · .fi-saveall

작은 흰 글래스 알약(10.5px). 진행 = 「저장 중… i/N」 글자. 완료 = 텍스트 "✓"(스타일 없는 글자 체크) 1.8초 뒤 원복.

↑ 탭 = 현재 동작

첨부2 검색이미지 캐러셀 「전체」 · .sbtn.cref-dlall

글래스 둥근사각(11px). 진행 표시 없음(바로 순차 저장). 완료 = 초록 채움 + ✓ 체크(CHECK_SVG) 1.5초 뒤 원복. ← 이 완료 체크-액티브가 더 또렷.

↑ 탭 = 현재 동작

통합 첨부3 — 한 디자인 + 첨부2식 ✓ 완료

두 버튼을 하나로: 글래스 알약(.sbtn 톤) + 진행 「저장 중… i/N」(첨부1) + 완료 초록 채움 + ✓ 체크(첨부2). 두 자리(섹션·캐러셀) 모두 이 한 벌로 통일.

↑ 탭 = 통합안 동작(진행 → 초록 ✓)
