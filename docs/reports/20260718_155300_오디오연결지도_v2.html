⚠️ 딱 2개만 확인

① JBL에 3.5mm 구멍(AUX) 있나?
뒤·옆·바닥 고무덮개 열어 이어폰 구멍 확인. 없으면 그 스피커만 블루투스(=KVM 자동전환 안 됨). 모델명 주면 바로 확정해줌.
② 윈도우에 USB 업스트림 선 꽂혔나?
윈도우는 DP만 꽂으면 소리가 안 넘어와. PC USB → 모니터 USB-C KVM 업스트림 선이 있어야 KVM 전환 때 소리도 같이 감. (맥은 썬더볼트가 알아서 함)

🎛️ 볼륨 (조용하게)

컴퓨터 80~100% 고정 → 스피커 자체 50~60% 고정 → 실제 조절은 HA400 노브. 맨 끝(스피커)을 키우면 앞단 잡음도 같이 커지니까, 끝은 중간에 두고 앞에서 조절 = 제일 조용.

✅ 소리길: 컴퓨터 → KA11(모니터 잡음 우회) → HA400(분배·음량) → 스피커 2대 · 새로 살 건 이미 산 AUX선 1개뿐
