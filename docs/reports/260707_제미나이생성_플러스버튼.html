장점

  • 타이핑 0 프롬프팅 — 옵션 버튼(화풍·비율·해상도·무드·문구)이 Opus에게 넘기는 지시가 됨. 최근 조합 자동 복원(로컬스토리지).
  • 한글 문구 렌더 지원 — 따옴표 리터럴·앞쪽 배치·긍정형 무결성 지시·2K 하한(UI 딤 + 서버 플로어 이중) = 아이데이션 분신술 조사 반영.
  • 기존 기능 무손실 — 실사진 재검색은 팝업 안 보조 버튼으로 그대로. 백엔드 실패 시에도 결정형 폴백 프롬프트로 렌더 강행(fail-soft).
  • 디자인 신규 raw 0 — pubdlg 셸·.pub-vis 워드칩·asksend 버튼·토큰 계승(check_refs 디자인 게이트 초록).
