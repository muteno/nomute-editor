효율성·비용

  • LLM 토큰 0 유지 — 순수 CV(SAM2·YOLO 로컬). 과금 = Actions 분 + R2 그대로.
  • 분석 1회 = 렌더 N회 불변 — 같은 분석으로 모자이크·핀셋·키잉 자유 재렌더.
  • 헤비 의존(torch+ultralytics ≈1.6GB)은 키잉 렌더 캐시로 격리 — 분석·모자이크 러너 복원 비용 불변.

품질저하 리스크 (정직 · 지침 3.b-1 등재)

  • 512 마스크 엣지 = 업스케일 계단 → 기본 페더 3px(게이지 0~40 조절). 로토 대체 수준 — 머리카락 낱올 스튜디오 매팅 아님.
  • 직접 지정 = 찍은 프레임부터 뒤로만 전파(앞 구간 미커버) — 스크럽에 "처음 나오는 장면에서" 힌트.
  • 세그 15fps 홀드 = 급격한 모션에서 엣지 1프레임 지연 가능.
  • 마스터 MOV 용량 큼(10초 200MB급 · 소재 따라 변동) — 폰 저장 부담. R2 유실 시 프리뷰만 제공(master-lost 정직 표기).
  • iOS 사파리 = VP9 프리뷰 재생 제한 가능(저장용 MOV는 무관).

검증 요약

  • 로컬 E2E: 분석(피사체 5카드 — 얼굴연결 2·뒷모습 2·버스 1) → 키잉 렌더(10초 = 130초) → 알파 픽셀 분포 실측 통과.
  • YOLO 디코드+NMS = ultralytics 정답과 5박스 수px 일치(정확성 게이트).
  • 헤드리스 5단계 실클릭(카드 토글·스크럽 지정·모드 왕복 선택 유지·결과 체커보드) JS 에러 0.
  • 실버그 1 자가 적발·봉합: 렌더 폴이 마스터 URL만 검사 → 프리뷰-only 결과 영영 미수신 → url‖preview 수용.
