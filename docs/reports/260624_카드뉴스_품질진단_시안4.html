🃏 카드뉴스 품질 진단 + 시안 4

요약(좋음) vs 카드(약함) — 분신술 11인(진단 10 + 종합 1) · 같은 기사 6쌍 실측

1. 진단 — 한 뿌리

카드가 약한 건 포맷 탓이 아니라, "아주 좋다"고 한 자유요약의 서사 척추를 카드가 구조적으로 안 물려받기 때문. 지침이 01_지침 L342 "자유요약은 입력 아님"이라 못 박아서, 카드는 좋은 요약을 옆에 두고도 못 쓰고 원문에서 0부터 재파생 → 인과·연결·전환·착지가 매번 얇게 다시 만들어진다.

결정적 증거 — 같은 룰·같은 압축캡인데:

✗ 상속 안 한 카드

올다르크: 합의는 이미 끝나 있었는데 → 왜 충돌인지 설명 없이 매달린 채 다음 카드로. 6장이 "제목 6개 나열".

✓ 요약 상속한 카드

ACC·부동산: 요약 줄기를 이어받아 起承轉結이 삶. 운전대 책임은 사람의 몫 = 이 기사 고유 착지.

→ 갈림길은 포맷이 아니라 "요약을 상속하느냐". 같은 .md 안에 좋은 자산이 이미 있는데 카드만 못 쓴다.

2. 왜 끊기나 (증폭 원인)

H1·주범
요약 비상속(L342) — 카드가 요약의 줄기·서사비트·시사점을 입력에서 배제, 원문 재파생.
H2
물리 압축캡(L355·361: 4줄·18자≈카드 72자) — 요약 한 문단(150~250자)의 인과 종속절("~지만/~사이/~데도")이 안 들어가 접속어만 남고 인과절은 삭제.
증폭
넘김훅 절단 기준(L356) — "긴장 지점에서 자르라"가 의미(인과)가 아니라 긴장 단위로 절단 → 비트 중간이 끊김.
메타
티어 비대칭(L406·407) — 조각화 룰(가중폭·글자수)은 강한 KILLER 티어, 흐름·서사·공감 룰은 약한 RECOMMENDED 티어 → 충돌 시 조각이 흐름을 항상 이김.
H3
마지막 카드(L340 원칙3) — "사회 메시지"가 일반 격언으로 빠져("법이 보는 건 행위다") 요약 💡의 기사 고유 시사점이 증발.

3. 고를 시안 4개

A. 자유요약 상속형최소수술·최대효과

핵심 변경 — L342 "자유요약은 입력 아님" 폐기 → 요약의 서사 척추(🔎줄기·📍서사비트·💡시사점)를 카드 골격 1차 입력으로 상속(원문은 사실 교차검증용). IG 📍비트 ↔ 카드 N장 1:1 매핑.

올다르크 카드1·2
전합의는 이미 끝나 있었는데 (왜 충돌인지 설명 없이 매달림)
후합의는 이미 끝났는데 / 그가 문을 막아섰다 → (카드2) 2명씩 들어가기로 / 모두 뜻을 모았지만 / 그를 못 돌려세웠다 (모순을 그 카드 안에서 닫고, 다음 카드가 이어받음)
장점
이미 검증된 자산 재사용 → 품질 상한이 요약 수준으로. 변경점 적음. 잘 된 카드가 이미 이 방식이라 회귀 최소.
단점
기틀 변경(설계 위계) = 승인+분신술 필수. 요약 편향까지 상속 위험(원문 사실가드로 상쇄).
B. 기승전결 템플릿 강제형편차·하한 잡기

핵심 변경 — 4-A에 起承轉結 슬롯 의무(轉=요약의 꺾임 문장 "그러나/문제는"이 든 카드 1개 지정). 흐름룰(#5·#30)을 RECOMMENDED→KILLER 승격(티어 비대칭 해소).

정점식 카드3·4
전결국 사퇴가 아니다 / 방점이 찍혔다 → (카드4) 그사이 회동은 계속됐다 / 결론은, 없었다 (제자리 맴돌이)
후결론도 시점도 비워둔 채로 → (카드4·轉) 그러는 사이, 회동만 늘었다 / 매듭은 없었다 / 책임은 누가 지나 (공전을 轉으로 터뜨려 착지로)
장점
편차를 게이트로 잡아 하한 ↑(같은 룰서 통과/실패 갈리던 분산 해소). 동화책 기승전결 구조 보장.
단점
템플릿이 사건 결을 평탄화할 위험. KILLER 승격 = 카드 캐시 버스트(미촬영 재생성). 억지 轉은 신파.
C. 시사점 역설계형착지 정조준

핵심 변경 — 마지막 카드 = 요약 💡의 이 사건 고유 논리(고유명사·수치·핵심명제)를 압축, 일반 격언 금지. 자가테스트: "다른 기사에 붙여도 말 되면 일반론=실패".

올다르크 카드6
전법이 보는 건 동기가 아니라 행위다 (아무 기사에나 붙는 법언)
후제도를 지키자며 / 제도를 멈춰 세운 두 시간 / 같은 행동에 진영은 / 영웅과 범법자를 붙였다 (이 사건만 성립)
장점
불만 ③(시사점 애매) 정조준. 변경 범위 국소(마지막 카드+💡 입력). 💡는 이미 잘 뽑혀 재사용 효율 ↑.
단점
착지만 고치면 본문 끊김(①⑤)은 그대로 = 불만 절반만 해결. A 없이 단독이면 본문 약한데 결말만 묵직 = 부조화.
D. 통합 단계형★ 권장A뿌리 + B흐름게이트 + C착지

핵심 변경 — A(요약 상속)를 뿌리로 → B의 흐름 게이트(#30 카드 대칭 적용)만 채택(轉 슬롯은 '권장'으로 완화=평탄화 방지) → C(고유 시사점 착지). 4-C STOP 직전 "요약↔카드 비트 커버리지 자가표" 1회 의무(추가 호출 0).

안락사 카드3·4·6
전벨기에는 나이 제한을 없앴다 → 캐나다는 더 무거웠다 / 가난과 돌봄 부족이 죽음의 이유로 (왜 무거운지 인과 비약)
후경계는 한 방향으로만 넓어져 왔다 / 벨기에는 나이 제한을 없앴고 → (카드4) 같은 문이 더 열린 캐나다선 / *가난*이 죽음의 이유로 → (카드6) 문은 약자에게 먼저 닿는다 (줄기가 꿰고, 이어받고, 고유 착지)
장점
불만 ①~⑤ 전부 커버(맥락·기승전결·시사점·사실력). A효과를 B·C가 하한 보강. 자가표로 드리프트 차단.
단점
변경 범위 최대 = 기틀+분신술 10인+캐시 버스트 동시. A→C→B 단계 롤아웃 권장(한 번에 다 X). 검토 부담 ↑.

검증 = 분신술 적대 Opus 4.8 11인(진단 10 + 종합 1)·약 1.44M 토큰·요약↔카드 6쌍 실측·실제 문장 인용 근거. 어느 시안이든 카드 텍스트 *룰* 변경 = 카드 캐시 버스트(미촬영 카드 텍스트 1회 재생성·제미나이 0·촬영완료 보호). 기틀(설계 위계 L342) 변경이면 §기틀 보호 승인 + 분신술 10인 후 머지. 정본 대상 = apps/news/01_지침_에디터_뉴스 Step 4.
