버튼 정렬 근본 + 프리셋 다양

글자 삐뚤어짐 = 근본(중앙정렬)으로 한 방에 잡음(모든 버튼 공통 룰). 아래 완성 프리셋 12종 중에 골라 — 세세한 조정 말고 느낌으로. 분홍선 = 박스 정중앙(글자가 여기 맞으면 안 삐뚤).

생성 버튼 미리보기

분홍 = 박스 정중앙 · 근본이면 글자가 선에 딱, 현행이면 위로 뜸

버튼 룩 프리셋 (느낌으로 골라)

전부 정렬은 근본(중앙) 적용 · 높이도 공통. 모양·톤만 다름.

높이(얇기)

  • 근본 정렬 = display:grid; place-items:center; line-height:1 — 계승 버튼(.asksend) 전체에 넣으면 요약전송·수정·발행·생성 어디서도 안 삐뚤어짐(개별 땜질 0). 원하면 이걸 전역 적용할게.
  • 프리셋은 전부 viewer 팔레트 안(accent·중립·글래스)이라 브랜드 안 벗어남.
  • 고른 프리셋 선택값 복사 → 생성 버튼에 반영하고, 원하면 레포 전반 버튼도 이 톤으로 통일할게.
  • 색·간격 = viewer :root 실측 · 폰트 시스템 폴백(실서비스 Pretendard).
