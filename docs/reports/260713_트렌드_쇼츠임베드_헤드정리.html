1. 무엇이 바뀌었나 (쉬운 설명)

  • 브리프 참고자료 팝업: 유튜브 쇼츠·틱톡·릴스·유튜브 링크면 요약 글 바로 아래에 영상이 인라인으로 뜬다. 쇼츠는 9:16 세로 원본 비율(높이 캡·중앙), 유튜브 일반은 16:9로 팝업 폭에 맞춤. 닫으면 재생·소리 즉시 정지.

2. 전 / 후 (실측 렌더)

전 — 팝업(텍스트만)
쇼츠 ref인데 요약 텍스트뿐 — 영상은 새탭 원문으로만.
후 — 쇼츠 = 9:16 원본비 인라인
요약 아래 세로 임베드(중앙·62dvh 캡). 회색 = 샌드박스 외부망 차단 자리 — 라이브에선 유튜브 플레이어가 재생됨.
후 — 유튜브 일반 = 16:9 폭맞춤
watch URL은 16:9 비율로 팝업 폭에 맞춰 요약 아래 표시.
전 — 트렌드 헤드(한반도 픽토·래퍼 잔존)
래퍼는 투명이지만 DOM 계층으로 잔존 + 국내 아이콘 = 한반도.
후 — 태극 픽토 + 래퍼 삭제
국내 = 태극(레드·딥블루 토큰 · on=원색 / off=무채 dim) · 최하단 블록 1층 제거 · 카드·레일·TOP10 렌더 정상.
