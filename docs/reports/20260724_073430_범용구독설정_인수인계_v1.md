## 3. 범용 구독·큐레이션 설정 재편 — 핵심 (현재 진행 중)

### 배경/의도 (운영자 260723~24)
구독 설정을 **게시판형 범용 메뉴**로 재편: ① 인스타·스레드도 큐레이션이라 통합 ② 뉴스/쇼츠/AI영상 **하드코딩 해체 → 설정에서 편집**(현재값=바꿀수있는 기본값) ③ 채널 아이콘=순번(SNS요약 순번제) ④ 채널 펼치면 세부분류 중첩.

### 실측 정정 (중요 — 운영자 추측과 다름)
뉴스/쇼츠/AI영상 = **특정 계정 아님, 키워드 검색 세트**:
- 뉴스 = YT 공식 **카테고리 25**(뉴스·정치) · 쇼츠 = `IT_QUERIES`(먹방·브이로그·예능 웃긴 영상·뷰티 메이크업 패션·영화 드라마 리뷰·여행) · AI영상 = `AI_QUERIES`(AI 영상 제작·AI 영상 생성·sora ai video·runway kling veo)
- 스크래퍼 정의 = `scraper/sns_trends.py` L148-149.

### 스키마 (하위호환 · 채널 스코프)
`viewer/sns_accounts.json` `youtube` 키에 계정과 **동거**:
```json
"youtube": { "kr": [핸들], "gl": [핸들], "shorts": [키워드], "aivid": [키워드], "news_cat": 25 }
```
- 미설정 키 = 스크래퍼/모달이 하드코딩 폴백 사용(= 종전 동작). `_load_accounts`(scraper)·`cleanPlat`(API)은 kr/gl만 읽던 것을 확장 = 무충돌.
- 타 플랫폼(x·tiktok·insta·threads) = 기존 {kr,gl} 그대로.

### A. Phase 1 — 스크래퍼 config화 ✅ 머지(#2891)
