사진첨부 버튼 글래스 통일 — 전/후

260628 · 운영자 "쟤 혼자 디자인이 따로놀아서 통일" · .askattach(뉴스요약 사진첨부) · PR #1148

핵심한 줄

주변 글래스 버튼(클립 .iobtn·표준 .sbtn)은 전부 backdrop blur(글래스 질감)가 있는데, .askattach만 flat(blur 없음)이라 혼자 떠 보였다. 배경·테두리는 이미 .iobtn과 똑같았고, blur 한 줄만 빠져 있었음. → 추가해서 통일.

전flat (아래 알록달록 배경이 그대로 또렷 = 유리 아님)

BEFORE · flat

빨간 점선 = 사진첨부. 클립·전송은 뒤 배경이 뿌옇게 갈려(유리) 보이는데, 사진첨부만 배경이 또렷이 비쳐 질감이 다르다.

후glass (사진첨부도 뒤가 뿌옇게 = 형제와 동일)

AFTER · glass

초록 점선 = 사진첨부. 이제 세 버튼 모두 같은 유리 질감(뒤가 동일하게 뿌옇게 갈림).

코드diff (1줄 추가)

코드
전.askattach { … background:rgba(255,255,255,.06); border:1px solid var(--line); border-radius:10px; … }
후.askattach { … 동일 … -webkit-backdrop-filter:blur(var(--blur-s)); backdrop-filter:blur(var(--blur-s)); … }

대조왜 '따로놀았나' (형제 비교)

버튼글래스 blur배경테두리
.sbtn (표준)blur(8px) ✓rgba(8,15,11,.3)rgba(255,255,255,.14)
.iobtn (클립·같은 모달)blur(var(--blur-s)) ✓rgba(255,255,255,.06)var(--line)
.askattach 전없음 ✗ (flat)rgba(255,255,255,.06)var(--line)
.askattach 후blur(var(--blur-s)) ✓rgba(255,255,255,.06)var(--line)

배경·테두리는 이미 .iobtn과 완전 동일했음 → blur만 추가하면 끝(픽토그램·번호 다중첨부 배지·눌림 .85는 이미 표준).

※ 이 미리보기의 --blur-s:8px·색은 라이브 viewer :root 값 복사(일회성 스냅샷). 같이 머지된 기능픽스 2건(이미지 선택 항상 작동·ly ▶ 첨부 후에만)은 시각 아닌 동작이라 표로 갈음 = PR #1144·#1146.
