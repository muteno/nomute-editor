"그 위에 박스" 어색함 점검 → 결과

✅ 어색해지는 중첩 박스 없음. 이유:

  • 뉴스 요약 카드: 내부 요소(제목·요약·진행바)가 패딩 18px로 인셋 → 카드 모서리에 붙는 자식이 없어 24→16px로 줄여도 코너 충돌 없음. 진행바는 알약(999px)이라 무관.
  • SNS 카드: 소스칩=알약, 패딩 12–14px 인셋 → 14→16px는 미세 증가, 충돌 없음.
  • 레거시: 기준(16px)이라 무변경. 내부 후보카드(13px)·실패트레이(14px) 관계 그대로.

제안 (선택 · 미적용)

요약 팝업(모달)은 22px로 남겨둠(--r-modal). 카드(16px)와 살짝 다르지만 모달은 별도 레이어라 조금 더 둥근 게 자연스러움(전 앱 모달 공통). ⇒ 그대로 두는 걸 추천.
원하면 모달도 16px로 맞출 수 있음(말해주면 반영).
참고: 레거시는 패널(16px)+내부 카드(13px) 2겹 구조, 피드·SNS는 카드 자체가 최외곽(16px) 단일 구조. 모서리는 16px로 통일됐고 구조 차이는 유지(요청 범위 = 모서리만).
