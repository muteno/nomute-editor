카드 버튼 액티브 = 합성기(이미지 제작) 계승 + 유료 재확인

뉴스 요약 2단계(카드 프롬프트)·3단계(카드 이미지=유료 슛) 버튼의 누름 액티브를 합성기 '이미지 제작'(.go) 전송 상태머신(초록 풀필 → ✓ 체크)으로 통일. 문구는 그대로. 각 버튼을 탭해봐.

① 2단계 — 카드 프롬프트 「프롬프트 다시 생성」

전(前): 하단 가느다란 게이지바만. 후(後): 합성기처럼 초록이 좌→우로 꽉 채우고 ✓ 체크가 그려진 뒤 '생성 시작' 상태로 넘어감.

↑ 탭 = 새 액티브(풀필 → ✓)

② 3단계 — 카드 이미지 「카드 이미지 생성」 · 유료

유료라 재확인 절차가 중간에 들어감. 1번째 탭 = 노랑 '유료 결제 — 한 번 더 누르면 생성'으로 무장(3.5초 뒤 자동 해제). 2번째 탭 = 합성기 풀필 → ✓ 체크로 실제 발사. (오발 과금 방지)

↑ 두 번 탭해야 발사(1탭=재확인·2탭=발사)

③ 기준 — 합성기 '이미지 제작'(.go) · 원본

계승의 원본. 위 ①② 가 이것과 동일한 풀필+✓ 모션이 됐는지 비교용.

↑ 탭 = 원본 모션
