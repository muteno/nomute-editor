요약요청(ask) 타임아웃 근본수리 + 실패 재시도 버튼

260704 · PR #1552 (squash 98bb4e0) · 10인 평의회 + 재검증 5인 만장일치

1. 무엇이 문제였나

뷰어 ✨요약요청에 기사 전문("중국인, 몇 시간 연수 후 렌터카 허용")을 붙여넣었는데 실패 — 원문도 못 찾고 재시도도 불가라 매번 전문을 다시 검색해 붙여야 했다.

진짜 원인 = claude -p가 900초(15분) 타임아웃(exit 124). 로그의 MultiEdit matches no known tool은 무해한 노이즈였다. 붙여넣은 전문은 멀쩡히 저장돼 있었음(입력 문제 아님).

  • 폴오버·재시도 안 걸림 — 타임아웃은 출력이 비어 is_quota/is_transient 정규식이 못 잡음(둘 다 출력 텍스트를 grep).
  • 오분류 — 타임아웃인데 "내용 분석 결함(입력이 비었거나 불충분)"으로 잘못 안내.
  • 왜 15분? — 전문을 이미 줬는데도 image_sources 지침이 "전문이라도 WebSearch 7~10개·여러 번"을 강제 → 검색 루프가 15분을 먹음. + --effort max로 각 검색 턴이 느림.

3. 재시도 흐름

실패 항목 ↗(재시도)→ api/askget 전문 복원→ 요약창 재오픈(전문 채움)→ 전송→ 성공 시 옛 실패파일 정리(retryOf)

4. 변경 요약

파이프라인 ask.sh·analyze.sh — effort high · 타임아웃 600/900s · claude_failover_force 1회 + claude_reset_force_swap(쿼터 3계정 체인 예산 보존) · 데드라인 가드(배치 초과 방지) · rc≠124 백오프 가드(2회 상한) · 오분류 정정.

프롬프트 news-analysis.md §0·§52 — 전문이면 url검색·image_sources를 2~3개로 최소화(타임아웃 근본원인 제거 · 공유 SSOT라 analyze도 치료).

API functions/api/askget.js(신규) 실패 전문 온디맨드 조회 · submit.js 재제출 성공 시 옛 파일 정리.

뷰어 대기열 요약요청 실패 ↗ → 재시도 버튼(go-arm 레몬토큰) · askRetry가 closeQueueUser 제거로 history 경쟁 회피.

5. 효율성

  • 타임아웃 자릿수급 감소 — 전문일 때 검색 루프(15~25회 → 2~3회)를 붕괴시켜 근본 제거. effort high로 검색 턴도 빨라짐.
  • 워크플로 안전 — force 1회 + 데드라인 가드로 과부하 배치에서도 잡 timeout(60/90분) 초과 방지.
  • 쿼터 보존 — 타임아웃 계정전환을 기사마다 되돌려(reset) 쿼터 3계정 체인 예산을 잠식하지 않음.
  • 사용자 노동 0 — 실패해도 재검색·재붙여넣기 없이 1탭 재시도.

6. 품질 저하 리스크 주의

  • effort max→high — 요약·편향 균형 품질이 미세 저하할 수 있음(평의회 "중간·미측정"). 완화 = 모델 opus 4.8 유지 + 지침 강제주입 + 검색완화가 남은 예산을 요약에 씀. → 카나리아 1건 실측 필요(측정으로 뒤집히면 PIPE_SEARCH_EFFORT=max env 한 줄로 롤백).
  • 원문 링크 감소 — 전문일 때 검색 포기가 늘면 "원문 ↗"가 빈 요약이 늘 수 있음. 운영자 우선순위(요약 완성 > URL·이미지)로 의식적 허용.
  • stuck-retry 정리 — 워크플로 처리 중 파일 삭제는 racy → best-effort(최악=현 상태, 악화 없음).

7. 검증

10인 적대 평의회(Opus·읽기전용) → ❌2건(타임아웃 45분 워크플로 초과 · 재시도 후 옛파일 24h 잔존) + ⚠️ 다수 발견 → 전부 반영 → 재검증 5인(force/reset·timeout루프·submit·history·검색완화) 만장일치 수렴.

기계 게이트 check_refs rc=0(폴오버·--bare·디자인 토큰) · 병합 후 git show origin/main read-back으로 실반영 확인. origin/main에 다른 세션이 넣은 간단수정(timeout_retried)은 이 포괄버전으로 흡수·통일.
