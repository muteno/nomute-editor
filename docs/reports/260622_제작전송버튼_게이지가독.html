버튼 진행모션 점검 — 제작·전송 버튼 가독성 수정

260622 · 대상 viewer/thumb.html(이미지 제작) + viewer/index.html(전송/수정) · 단일 데모 파일(이 보고서는 일회성 스냅샷)

🔎 한눈에

두 버튼 다 코드는 작동했지만(헤드리스 렌더로 확인) 진행 모션이 사람 눈엔 안 보이거나 이상했다. 원인은 서로 달랐다:

  • 이미지 제작 — 게이지가 하단 3px·0.72s 1회인데 라벨 변화가 없어 너무 약함(모션 줄이기 켜면 0.2s 섬광). 실제 작업은 9초+라 버튼은 곧 정지.
  • 전송 — 어두운 게이지 위에 글자도 어두워 전송 중 "전송" 글자가 사라지고 검정 덩어리로 보임. 코드 주석은 "PICK과 동일한 초록 게이지"라는데 실제론 색이 정반대로 구현된 불일치 버그.

→ 둘 다 기존 디자인 패턴을 계승해 수정. 관련 버튼 전수 점검도 완료(아래 표).

💡 아래 버튼들은 실제로 눌러볼 수 있다(viewer 정본 CSS 그대로). 전/후를 직접 비교해봐.

📍 1. 이미지 제작 버튼

전 (Before)

▶ 눌러봐 — 하단 3px 게이지 한 번 깜빡, 라벨 그대로

후 (After)

▶ 눌러봐 — "전송 중…" → "전송됨 ✓" 초록 채움

라벨이 바뀌어서 모션 줄이기를 켜도 무조건 보인다(텍스트는 애니메이션 무관). 게이지는 그대로 두고 가이드북 전송 상태머신(대기→전송 중…→완료✓)을 계승했다. "완료"가 아닌 "전송됨"인 이유 = 실제 이미지 제작은 9초+ 걸려 아래 작업카드가 진행을 표시(버튼은 *전송*만 책임 = 정직).

📍 2. 전송 버튼 (요약요청 · 수정요청)

전 (Before)
보내는 중…

▶ 눌러봐 — 어두운 게이지가 덮으며 "전송" 글자 소멸 → 검정 덩어리

후 (After)
보내는 중…

▶ 눌러봐 — 글자 초록으로 뒤집혀 끝까지 읽힘

어두운 와이프는 밝은초록 버튼 위 고대비 게이지로 오히려 좋다 — 문제는 글자색을 안 뒤집은 것뿐. color:var(--accent) + 전송중 opacity:1(dim 해제) 두 줄로 해결. .asksend·.revsend 둘 다 적용.

📍 3. 관련 버튼 전수 점검 (firing 패밀리)

버튼게이지글자색판정
이미지 제작 .go하단 3px 초록바—수정 = 라벨 상태머신 추가
전송 .asksend어두운 와이프어두움수정 = 글자 초록 뒤집기
수정 → .revsend어두운 와이프어두움수정 = 동일 버그·동일 처리
PICK .sc-pick초록 와이프어두움정상 (정본 레퍼런스)
카드 슛 .mkbtn하단 3px 초록바안 덮음정상
카드 수정 .ed-go초록-on-초록(은은)어두움+라벨"전송 중…"정상 (라벨로 표시)
재분석 보내기 #pasteSend없음(상태텍스트만)—정상 (소멸 없음·단순형)

깨진 건 어두운 와이프 위 어두운 글자를 쓴 전송 패밀리 2개뿐. 나머지는 글자색이 게이지와 대비돼 멀쩡. 일관 규칙 = "어두운 와이프엔 밝은 글자 / 초록 와이프엔 어두운 글자".

📍 4. 분석 (전후·장단점·효율·리스크)

전 → 후
제작: 3px 1회 섬광 → "전송 중…→전송됨 ✓" 라벨 상태머신.
전송: 글자 소멸 검정덩어리 → 초록 글자 끝까지 가독.
장점
· 라벨/색 변화라 모션 줄이기에도 보임.
· 기존 패턴 계승 = 재설계 0·일관성↑.
· 제작 버튼 헛발사(검증 실패 시 게이지) 잔버그도 동시 제거.
단점/트레이드오프
· 제작 버튼 피드백이 ~1.7s로 길어짐(의도 — 읽히게).
· "전송됨"이 가이드북 표준 "완료"와 글자 다름(정직 위해 의도적).
효율
· 순수 CSS/JS·LLM 0·과금 0.
· GPU 컴포지트(transform/opacity) = 60fps·부하 무시.
· 변경 = thumb.html 함수 1개+CSS 1줄, index.html CSS 2줄.
⚠ 품질저하 리스크 — 낮음. ① 셋 다 국소 뷰어 손질(:root·배너·파이프라인 무관). ② check_refs(참조·JS구문·디자인거울) 통과. ③ 헤드리스 렌더로 전/후 픽셀 확인. ④ 연속 전송·재진입은 타이머 클리어로 한 흐름 유지. 남은 변수 = 실제 폰 터치 시 손가락 가림(버튼 하단 게이지) — 라벨 변화로 보완됨.
