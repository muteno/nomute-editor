모든 모달·팝업은 이 셸 하나를 계승한다(재설계 금지). 핵심 미감 = 글래스모피즘: 반투명 배경 + backdrop-filter: blur로 뒤 콘텐츠가 은은히 비쳐 층이 진다. 헤더 띠는 내용보다 살짝 도드라지고, 내용은 컨테이너 글래스가 비쳐 물러난다. 아래 데모는 실제 backdrop-filter로 렌더 — 뒤 컬러 블롭이 뭉개져 비치는 게 글래스다.

계승 규칙 (신규 팝업 = 이것만 끼운다)

  • 헤더 띠 = .modal-head 정본(별칭 .tool-h/.dlg-h/.ed-h/.mh/.qh/.pmenu-h/.pin-head). 배경 --modal-head-bg + blur --blur-l + border-bottom --line2.
  • 탭 띠 = .modal-tabs 정본(별칭 .tool-tabs). 배경 --modal-tabs-bg · has-tabs일 때만.
  • 컨테이너 = 중앙 dialog(딤 있음) --modal-glass(.64) / 앵커 div(딤 없음) --modal-glass-anchor(.75).
  • 내용 = 투명(배경 없음) → 컨테이너 글래스가 비쳐 층이 진다.
  • 앵커 팝업 전폭 = 헤더에 margin:-컨테이너padding 상쇄 + radius calc(--r-l/--r-modal − 1px).
  • 예외 — 불투명 컨테이너(ask-family) = blur 생략, border-bottom + 제목 톤만(동색이라 띠 안 보이고 260701 번쩍 회피). 이것도 반영됨(border+톤).
  • raw 창작 금지 — 값은 :root 4토큰만. 새 값 필요 시 운영자 승인 → 토큰 신설 + 거울 + CII + lock.
