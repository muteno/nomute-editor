바뀐 점 · 재현 한계

  • 고정 리드 = 내용 소머리 + 주문칸 + 우측하단 선택요약 + 생성 버튼 → 스크롤 내려도 상시 상단 고정.

  • 재현 한계: 실 #genidlg는 backdrop-blur·아코디언 촤르륵 모션·드롭다운 선택박스·무드 게이지 4축이 더 있음(여기선 정적 근사). 색·간격·radius는 viewer :root 실측 사본. 폰트는 시스템 폴백(실서비스 Pretendard).
