레포에 통화/코인 전용 픽토그램이 없어서 신규 제작이야. 문제는 XRP·SOL은 표준 유니코드 심볼이 없고, ₿·Ξ·元도 폰트(Noto Sans KR) 표시가 안 보장돼 — 그래서 "글리프로 넣기 vs SVG 제작"이 갈려. 아래 토글로 실제 금융 행에 얹어 비교해봐. 색·마진·칩은 전부 정본 토큰 계승.

3안 요약
· 안 A 텍스트 심볼 — 원형 칩에 통화기호($ € ¥ ¥) + 코인(₿ Ξ · XRP/SOL은 폴백문자 X·◎). 가장 빠름. 단점: ₿·◎ 폰트 미표시 리스크, XRP는 심볼 부재라 대용 문자.
· 안 B 모노그램 — 원형 칩에 통일 문자(달러 $, 유로 €, BTC=B, XRP=X…). 100% 안정·통일감. 단점: 심볼보다 인식성 약간 낮음(B가 비트코인인지 학습 필요).
· 안 C SVG 픽토그램 (추천) — 통화·코인 마크를 SVG로. 폰트 무관 100% 표시·§3-1(도형=SVG) 준수·최고 인식성. 아래 샘플은 초안이라 채택 시 획 다듬기 + §3-4 픽셀 중앙정렬 실측(십자선 증빙) 후 반영.
공통 — 칩은 --r-pill 원형·--mut·rgba(255,255,255,.06) 계승, 행 baseline 유지하고 칩만 align-self:center로 광학 중앙높이정렬. 이름(미국 달러·BTC)은 그대로.
기준점 캡션은 이미 반영됨(「· 07/17 00:00 대비」 = 오늘 자정 기준).
