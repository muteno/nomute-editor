01네이버 카페/블로그 → 픽토그램 대체#1533
전 → 후
"네이버카페"·"네이버블로그" 긴 키워드 칩 → 카페=☕컵 픽토 + "카페", 블로그=✒️펜 + "블로그", 지식iN=텍스트만. 색은 약진보 시안 동축.
리스크
픽토그램이 작아(11px) 저해상 폰에서 컵/펜 구분 모호 가능.
장단점
＋칩 폭 절반↓·시각 정돈. －아이콘만으론 처음 보는 유저에 학습 필요(그래서 짧은 라벨 병기).
단점 보완
픽토 + 짧은 텍스트 라벨 병행(아이콘 단독 아님) = 학습부담 상쇄.
추후 계획
타 소스(디시·에펨 등)도 픽토 도입 여지 — 현재는 네이버 계열만.
아이데이션
지식iN도 전용 픽토(전구·Q) 검토 가능하나 텍스트가 이미 짧아 보류.
신의 한수
색(성향)은 시안 그대로 두고 플랫폼 구별만 픽토가 담당 = 성향축과 플랫폼축을 분리해 둘 다 안 깨짐.
이건 모르겠다
"카페=컵" 은유가 한국 유저에 직관적인지 — 카페(커뮤니티)와 카페(음료)의 말장난이라 호불호 갈릴 수 있음.
