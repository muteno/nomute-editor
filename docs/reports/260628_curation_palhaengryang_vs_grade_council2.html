① 개념 재정의 — 발행량축 ⊥ grade축 (직교, 곱하면 안 됨)

┌─ 발행량축 (X) = 경중·대세·순위 ──┐ ⊥ ┌─ grade축 (Y) = 긴급·알림 ─┐ │ "전국민이 얼마나 보고 있나" │ │ "지금 터진 사고·재난인가" │ │ 신호: cross · report_count · burst │ │ 신호: AI grade(0~3) │ │ 객관·기자주도·검증됨 │ │ isBreaking(brk&g≥2) │ │ → 누적 칼럼 세로 순위를 정한다 │ │ isAlert(brk&g≥3)→푸시·배지 │ │ grade는 여기 안 들어온다 │ │ 발행량은 여기 안 들어온다 │ └────────────────────────────────────┘ └──────────────────────────────┘

핵심 명제: grade는 "이 사건이 *지금 알림 쏠 만큼 긴급한가"의 이진 트리거다. "이 기사가 *얼마나 무거운가"라는 연속 경중 랭킹과는 다른 질문. 둘을 곱한 게 죄.

🔴 정형 잡음 (가장 중요·#3·#7)

증시현황("환율 1537.4원에 출발")·[특징주]·[마감시황]·운세·날씨 ~79건은 발행량으로도 연속보도로도 안 걸러짐.

충격 반전: rc가 거꾸로다 — 운세 rc54·날씨 rc31·특징주 rc18 (정기 자동발행이라 폭증) > 홍명보 대세 rc1~3. "rc가 보도자료를 가른다"는 가정 데이터로 반증.

→ JUNK 머리표 게이트([특징주]·[마감시황]·[날씨]·[운세]·[인사]·[부고]) 동반 필수. 이미 있는 JUNK_HEAD 앵커 강화.

🟡 계수 재조정 (#6)

gradeW 빼면 cross^1.6(30배 폭)이 지배, 연속보도(1.5배)는 묻힘 → cross17·rc1(한탕)이 cross5·rc14(연속보도)를 이김 = "연속보도 제일 무겁게" 원칙 위반.

→ cross 지수 1.6→1.3, 연속보도 강도 0.2→0.5(또는 rc 티어 정렬). 단 §★ 고정 임계라 별점 검증 후.

🟡 스포츠 단일사건 도배 (#4·#2)

발행량만 믿으면 홍명보 32강 1경기가 cross17 기사 20개로 상단 점령. → event 클러스터 캡(같은 사건 상위 N개만).

🟢 grade 긴급 역할은 보존 (#5·#1)

isBreaking(g≥2)·isAlert/푸시(g≥3) = 비가역 오발 안전핀. gradeW 안 거치고 raw grade 직독 = 물리적으로 분리 → 한 줄도 안 건드림.

⑤ 💡 외과적 통찰 — gradeW의 진짜 죄는 곱셈이 아니라 ×0.05 floor

전수 데이터 증명: grade0 floor를 ×0.05 → ×0.5로만 올려도 —

  • (a) cross16~17 축구 대세는 발행량으로 톱 복귀
  • (b) rc1~2 순수 보도자료는 여전히 낮은 scFollow로 침몰 유지
  • (c) 정형 잡음(운세·증시현황)은 JUNK_HEAD가 별도 컷

= "grade를 랭킹에서 빼라"의 가장 안전한 구현 = grade를 *연속 가중치*에서 *이진 게이트*로 강등. 긴급이면 배지(Y축), 잡음이면 약한 감점, 그 외엔 발행량(X축) 지배. 장기적으로는 grade 0~3 서열 폐기 → is_junk ⊥ is_urgent 이진화(#7)가 깔끔한 착지.
