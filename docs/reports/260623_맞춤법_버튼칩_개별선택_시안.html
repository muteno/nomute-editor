맞춤법 교정 — 버튼 + 칩 개별선택

샘플 = 속보 됬어요 함니다 외않되(오타 3). 이미지 제작 및 맞춤법 교정을 눌러봐 → 칩이 뜸. 칩을 탭해 점등을 켜고 끄며 버튼 색이 바뀌는 걸 봐. 한 번 더 누르면 적용+생성.

  • 1번째 누름 → 맞춤법 검사 → 바뀔 부분이 노란 칩(점등)으로 버튼 아래 뜸. 버튼은 노랑(재확인).
  • 칩 탭 → 점등 해제(그대로 둠) ↔ 다시 탭하면 점등. 개별 선택.
  • 다 점등 = 전부 교정 → 노란 체크 생성 · 일부만 점등 = 그것만 교정 · 다 해제 = 맞춤법 0 → 초록 진행.
  • 2번째 누름 → 점등된 것만 텍스트에 적용 + 생성. 적용 있으면 노란 체크, 없으면 초록 체크.
