① 바꾸는 것 (네 1~10 요구 반영)

  • 필터(카테고리 칩바) → 헤더에서 빼고 깔때기 픽토그램 버튼(테두리 X·픽토그램만)으로. 누르면 그 아래 모피즘 글래스 오버레이로 칩이 펼쳐짐. (점2·7)
  • 카드만 토글 → 카드 픽토그램 버튼(테두리 X)으로. 평소 무채색(off), 누르면 네온그린(on). (점3·9)
  • 두 버튼 모두 프로필 바로 아래 우측에 나란히. (점4)
  • 필터바가 빠져 생긴 여백으로 feed 첫 카드가 위로 올라와 3탭 첫 블록 라인 통일. (점1·5)
  • 전부 기존 토큰·컴포넌트 계승(.ts-toggle 무테 아이콘 · msgpop 글래스 · popIn · :root 토큰). 새로 만든 건 픽토그램 2개뿐. (점10)

② 인터랙티브 목업 — 직접 탭해봐

아래 폰에서 깔때기=필터 오버레이 열림 / 카드=네온그린 토글. BEFORE/AFTER 버튼으로 비교.

3탭 정렬 비교 (AFTER)

초록 점선 = 첫 모피즘 블록 시작선. AFTER에선 3탭 모두 같은 라인(점선)에서 시작.

③ 레이어(z-index) 처리 보고 (점7)

필터 오버레이가 프로필 팝업(메뉴·메시지함)과 같은 우상단에서 열려서 겹칠 수 있음. 처리 방식:
