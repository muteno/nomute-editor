대기열 — 처리중/재시도 스피너 무도형 좌측정렬

요청대로 반영 완료(라이브 main). 스피너를 도형(박스) 밖으로 빼서 완료·실패 한글처럼 좌측정렬로 통일했어. 재시도 횟수 숫자는 재시도 스피너와 같은 앰버색이라 박스가 사라지니 자연히 한 묶음으로 맞음. 스피너는 실제로 돌아가(아래 미리보기 라이브).

상태 4종 정리
• 처리중 = 앰버 스피너만(도형X) · 분석 진행 중
• ↻ 재시도 + 숫자 = 앰버 스피너 + 앰버 횟수(2) — 숫자색 = 스피너색(앰버) 동일(요청2 충족)
• 완료 = 초록 한글 + 초록 ↗(뉴스요약 열기)
• 실패 = 빨강 한글 + 앰버 ↗(원문은 열림) ← 실패 시안

핵심: 이제 완료·실패 한글과 처리중·재시도 스피너가 모두 같은 좌변에서 시작(↗ 화살표 위치도 행마다 일정). 혹시 스피너가 글자보다 커 보이거나 숫자색을 더 진하게/연하게 원하면 말해 — 바로 미세조정할게.
