키워드 칩 — 복사 누름 피드백 (목업)

지금은 탭하면 "복사됨 ✓" 글자로 바뀌었다 돌아옴. 대신 잠깐 눌린 것처럼 표현했다 복귀하는 5안. 각 칩을 실제로 탭해봐 (실기기 권장 · 반복 탭 가능). 마음에 드는 번호 알려주면 그걸로 반영할게.

A눌림만 (스케일)

제일 담백 — 살짝 쑥 들어갔다 스프링백. 색 변화 0.

검색 이미지
곡성, 물놀이, 테마파크, 합동감식, 감전
B눌림 + 초록 채움

누르며 초록(복사됨 색)으로 잠깐 차올랐다 빠짐. 글자는 그대로 — "복사됐다"가 색으로 전달.

검색 이미지
곡성, 물놀이, 테마파크, 합동감식, 감전
C인셋 그림자 (눌려 들어감)

물리 버튼처럼 안으로 쑥 — 그림자가 안쪽으로 들어갔다 복귀. 촉감 강조.

검색 이미지
곡성, 물놀이, 테마파크, 합동감식, 감전
D초록 링 펄스 + 눌림

누름과 동시에 초록 테두리 링이 한 번 퍼졌다 사라짐. 가장 "반응했다" 느낌 강함.

검색 이미지
곡성, 물놀이, 테마파크, 합동감식, 감전
E밝기 플래시

탭 순간 칩이 번쩍 밝아졌다 복귀 + 살짝 눌림. 화사한 피드백.

검색 이미지
곡성, 물놀이, 테마파크, 합동감식, 감전

참고 — 실제 반영 시엔 탭하면 클립보드에 키워드가 복사되고(지금과 동일), 화면 피드백만 위 모션으로 교체돼. 글자 스왑("복사됨 ✓")은 제거. 모션 끄기 설정이면 즉시(애니 없음).
