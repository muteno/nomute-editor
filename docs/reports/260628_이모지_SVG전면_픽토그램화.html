한눈에 (쉬운 설명)

화면에 박혀 있던 컬러 이모지(🚨⚠️✅🍌👍 …)를 전부 선(line) 픽토그램 SVG로 바꿨다. 왜 버그였나 — 이모지는 ① 기기·OS마다 모양이 다르게 그려지고(안드/아이폰/윈도우 제각각) ② 강조색(긴급=빨강)을 못 입히며 ③ 크기·정렬이 글자에 휘둘려 들쭉날쭉했다. SVG는 currentColor를 상속해 상태색을 그대로 입고, 어디서 봐도 같은 모양·같은 크기로 렌더된다. 동작·데이터는 한 줄도 안 건드렸다 — 보이는 글자(아이콘)만 교체.

2안전 설계 — "기존을 안 틀어버림"

  • 동적 문자열은 esc() 동반 — textContent→innerHTML 전환 시 서버 오류메시지(d.error·e.message)가 HTML로 안 새도록 이스케이프(XSS 가드).
  • 헬퍼는 하위호환 — qFlash(msg, icon)·edStatus(t, cls, html) 처럼 인자 추가로만 확장. 기존 호출부(텍스트 경로)는 그대로 작동.
  • currentColor 상속 — 새 SVG는 색을 안 박고 부모 색을 따름 → 긴급 빨강·평점 골드 등 기존 CSS 상태색이 그대로 적용.
