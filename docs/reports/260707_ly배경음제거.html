자막기 배경음 제거 — 목소리만 남기기

260707 · 운영자 확정: 컷 편집과 각각 독립 토글 · 둘 다 켜면 배경음부터 · 기본 OFF · Demucs 보컬 분리(로컬·과금 0)

① 무엇이 생겼나

  • 자막 설정에 배경음 제거 칩 — 켜면 완성 영상의 소리가 목소리만으로 바뀜(배경음악·환경음 소거).
  • 컷 편집과 자유 조합: 둘 다 켜면 배경음 먼저 지우고 → 무음 컷(순서는 코드 구조로 강제·역전 불가).
  • 분리 후 작아지는 체감 음량은 라우드니스 자동 보정(-16 LUFS = SNS 표준)으로 복원.

② 실측 검증(전부 실제 영상·실행)

항목결과
분리 품질(정량)말 없는 구간 −17.4 → −40.7dB(−23dB 감쇠) = 배경 거의 소거 · 발화 구간 −18.1 → −20.9dB = 목소리 보존
4조합 매트릭스둘 다 ON = 22.8초 + LUFS −16.5 · 배경음만 = 26.3초(영상 길이 보존) · 컷만 = 22.8초 · 둘 다 OFF = 종전과 동일(회귀 0)
처리 속도26.3초 영상 분리 18초(CPU) — 릴스/쇼츠 여유·10분급 장영상은 시간 초과 시 자동 스킵(원본 소리)
실패 안전망깨진 분리 결과를 강제 주입해도 원본으로 강등해 영상은 반드시 산출 + 노트에 "배경음 제거 실패" 정직 표기(실증)
평의회10인 적대 검증 — P1 6건 전부 코드 반영{영상 절단 회귀(-shortest) 제거 · 버전 핀 · 라우드니스 · 무효 분리 봉합 · 예산 백스톱 · 주석}

③ 전 → 후

축전후
BGM 깔린 영상자막만 얹음(소리 그대로)목소리만 남긴 완성본 원탭
음량—분리 후 -16 LUFS 자동 정규화
비용—과금 0(러너 로컬 모델·7일 캐시) · 러너 시간 +수 분

④ 한계·리스크(정직)

  • 분리 특성상 목소리에 약한 잔향/아티팩트 가능(음악 위에 말하는 영상에서 체감 가능성) — 첫 실사용 몇 건 확인 권장.
  • 10분급 장영상은 분리 시간 초과로 스킵될 수 있음(노트로 표면화 — 릴스/쇼츠엔 무관).
  • 다시 입히기(reburn)마다 분리를 다시 함 — 분리 결과 R2 보관(재사용)은 후속 후보.

⑤ 후속 후보

  • 목소리 강조 모드: 완전 제거 대신 배경 15%만 남김(분리 부산물 재활용 = 공짜 구현).
  • 필러 컷: "음·어" 어절 구간을 영상에서도 컷(운영자 아이디어 답변의 1순위 — 별도 진행).
  • vocals R2 캐시(재합성 가속) · STT 앞 분리(소음 영상 전사 품질).
