잠금 설정 — 「변경 전 기존 PIN 확인」 게이트 제안

260703 · 문제: 지금은 누구든 잠금 설정을 열어 PIN 교체·토글 OFF 가능 = 잠금 의미 반감. 아래 데모는 실제 팝업 CSS로 동작함 — 직접 탭해서 체험.

A안(추천) — 인라인 가드: 팝업 안에서 현재 PIN 먼저

PIN이 이미 설정된 기기에서 잠금 설정을 열면 → 설정 행들이 디밍·비활성되고 상단에 「현재 PIN 확인」 입력이 먼저 뜸. 현재 PIN 또는 마스터(898900) 통과 시에만 편집 활성(팝업 닫으면 다시 잠김).
