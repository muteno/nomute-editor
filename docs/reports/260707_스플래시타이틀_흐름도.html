OS 안드로이드 스플래시 타이틀 제거 — 케이스 흐름도

260707 · 진입(OS 스플래시) → 앱 인트로 페이드 교차는 이미 적용됨(#bootveil). 여기선 "타이틀 안 뜨게"를 3가지 방법으로 비교.
왜 케이스가 갈리나: OS 스플래시는 안드로이드가 직접 그림 — 배경색(manifest background_color #192730) + 아이콘(가운데) + 앱이름 텍스트(manifest name). 지금 아이콘 자체가 "News Curation" 글자가 박힌 워드마크라, 텍스트 라벨만 지워도 아이콘 글자는 남음. 그리고 WebAPK는 홈 런처 아이콘 = 스플래시 아이콘을 같은 파일로 씀 → 아이콘 바꾸면 홈 아이콘도 바뀜. ⚠️ 어느 케이스든 실기기 반영은 WebAPK 재민팅 1~3일(즉시 = 앱 재설치).
