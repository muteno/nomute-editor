⚠️ 사기 전 / 조립 전에 딱 2개만 확인

① JBL 사운드박스에 3.5mm 구멍(AUX) 있어?
  • 스피커 뒤·옆·바닥에 고무 덮개 열어서 이어폰 구멍(⌀ / AUX / LINE-IN) 있는지 봐.
  • 있으면 → 에디파이어랑 똑같이 젠더로 물리면 끝. 계획대로 OK.
  • 없으면(요즘 JBL 다수 = BT 전용) → 그 스피커만 블루투스로 가야 함. 단 BT는 KVM 자동전환이 안 돼서 PC 바꿀 때마다 재연결이 귀찮음.
  • 👉 모델명 알려주면 있는지 없는지 바로 확정해줌.
② 윈도우 PC에 USB 업스트림 선 꽂혀 있어?
  • 윈도우는 DP(화면)만 꽂으면 소리(DAC)가 윈도우로 안 넘어와.
  • PC USB → 모니터 USB-C KVM 업스트림 포트, 이 선이 반드시 있어야 KVM 전환 때 소리·키보드·마우스가 같이 넘어감.
  • 맥북은 썬더볼트 한 선이 이걸 다 해서 신경 안 써도 됨.

🛒 부품 현황 (뭐가 동봉이고 뭘 사야 하나)

KA11 → 컴퓨터 선KA11에 USB-C 꼬리선 일체형. 모니터 USB-C 포트에 바로 꽂음 → 추가 선 0. (C→A 어댑터는 동봉이 들쭉날쭉이라 아예 USB-C 포트 쓰는 게 깔끔)
KA11 → HA400 선3.5mm AUX 케이블 1m 삼 ✓ + 6.35 젠더 보유
HA400 → 스피커 ×26.35→3.5 젠더 2개 보유 — 젠더는 길이 0이라, 스피커 자체 선이 HA400까지 닿아야 함
맥·윈 ↔ 모니터썬더볼트4 / DP / USB 업스트림 기존 그대로

결론: 새로 살 건 이미 산 AUX 케이블 1개뿐. (HA400 아직 없으면 그것만 추가)

🔧 도착하면: 딱 한 번만 설정

  • 1맥: 시스템 설정 › 사운드 › 출력 = FiiO KA11 선택
  • 2윈: 소리 › 출력 장치 = FiiO KA11 선택
  • 3모니터 KVM 짝: 「DP입력 + USB-C업스트림 = 윈도우」 / 「썬더볼트 = 맥」

이후엔 KVM 버튼 하나로 화면·USB·소리가 통째로 전환. 전환 순간 잠깐 뚝 끊겼다 다시 잡히는 건 정상.

🎛️ 잡음 없는 볼륨 공식

① 컴퓨터(맥/윈)높게 80~100% 고정 (소리 갈라지면=클리핑, 그때만 조금 낮춤)
② 스피커 자체 볼륨중간 50~60%에 고정하고 손 안 댐
③ 실제 조절HA400 노브 1·2번으로 최종 음량 + 두 스피커 밸런스

이유: 맨 끝(스피커)을 키우면 앞단 잡음까지 다 같이 커짐. 그래서 끝은 중간에 두고, 앞쪽에서 조절하는 게 제일 조용함.

🔍 검증 요약 — 왜 이 구성이 맞는지 / 흔한 오해 정정

• HA400 쓴 거 잘한 선택. 그냥 Y자 2구 젠더로 쪼개면 신호가 약해지고 두 스피커 볼륨을 못 맞춤. HA400은 버퍼+채널별 볼륨이라 이 용도에 오히려 정답.

• "컴퓨터 볼륨 낮추면 데이터가 삭제된다" → 반은 틀린 말. 요즘 OS는 손실 거의 없음. 볼륨 높게 두는 진짜 이유는 '노이즈'(위 볼륨 공식 참고).

• "DAC는 모니터 말고 본체에 꽂아야 음질 좋다" → USB는 디지털이라 모니터 허브나 본체나 음질 동일. 모니터 허브에 꽂아야 KVM 자동전환이 되니 지금 방식이 맞음.

• 완벽판이 궁금하면: KA11(헤드폰출력)→HA400(헤드폰앰프)→액티브스피커 = 앰프가 3번 겹침. 동작·음질에 실제 문제는 없고 체감차도 거의 0. 완벽주의로 가면 라인아웃(RCA) 달린 거치형(Topping DX1류)이 한 끗 깔끔하지만, 네 목적엔 KA11로 충분 = 돈 더 쓸 필요 없음.
