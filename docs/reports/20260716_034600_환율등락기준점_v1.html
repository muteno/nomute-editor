데이터 기준점 — 환율 등락률 = Frankfurter(ECB 공시·무키) 시계열의 직전 거래일 종가 대비(주말·공휴일 자동 스킵). 표시 환율값은 기존 open.er-api.com 유지 — 등락률만 동일 소스 2일 비교로 산출해 소스 격차가 가짜 등락으로 새는 것을 차단. 암호화폐 등락률 = 업비트 전일 종가 대비(signed_change_rate). 두 축 모두 「전일 대비」로 통일 라벨.
