결론: 12시간 내 뉴스 레거시 경로를 깬 코드 변경은 없다. 수집·판정·데이터·화면 렌더 로직 전부 지금 정상 작동 중. 진범은 코드가 아니라 데이터 덩치 — candidates.json이 3.45MB(GitHub 1MB 한도의 3.5배)로 비대해져 라이브 서빙 api/candidates가 빈 []를 뱉고, 뷰어가 그걸 수용해 수집함을 통째로 비운다.
