OS 키보드 대신 자체 숫자 키패드. 6칸 슬롯 중 입력한 것만 강조색(빈 칸은 무채색), 키패드는 한 자리 누를 때마다 위치가 셔플돼(어깨너머·키로깅 완화). 6칸 채우면 자동 검증. 아래 직접 눌러봐 — 123456=성공 · 898900=비밀번호 초기화 완료(안내) · 그 외=실패(빨강+흔들림).

방식 (PAYCO 계승)

  • 6칸 슬롯 — 입력한 것만 강조색, 빈 칸 무채색
  • 셔플 키패드 — 매 입력마다 숫자 위치 재배치
  • 자동 검증 — 6칸 차면 바로(열기 버튼 X)
  • 지우기(⌫)로 한 자리씩 정정 · 데스크톱 물리 키보드도 허용
  • 실제 PIN은 hidden으로 GET 제출 = 서버 검증(셔플은 표시용)

상태 피드백

  • 확인 중 = 슬롯 펄스 + 패드 비활성
  • 성공 = 문서로 페이지 교체(자동 오픈)
  • 실패 = 슬롯 빨강 + 흔들림 + 처음부터
  • 마스터(898900) = ‘비밀번호 초기화 완료’(mut 회색·흔들림 X)
