음성 클로닝 구축 — Replicate RVC 배선

260712 · 벤치 판정(Seed-VC CPU 탈락 = 스텝당 506초) → 운영자 버튼 승인 → 학습·입히기 전 구간 배선 · Replicate 실호출 = 키 등록 후 카나리아로 확정(미검증 정직 표기)

전 / 후

전

  • 음원 탭 = 옵션 → 수노 프롬프팅 / 구글 곡 생성까지 (음색은 엔진 기본 보컬 고정)
  • 수노 Voices(클론) 축 = 대체 불가 판정 상태
  • pollSong 에러·타임아웃 표면화가 리팩터 회귀로 조용히 죽어 있었음

후

  • • 음성 클로닝 섹션: 소스(1~5분)+이름+동의 토글 → 보이스 학습 → 자동 선택
  • 완성곡·플레이리스트 재생 화면 → [보이스 입히기] → 내 음색 커버가 플레이리스트 합류
  • 수노 대체표 승격: Voices 축 = 부분 대체(내 보이스 한정) · Stems 2분리 원료(Demucs 35s) 확보
  • pollSong 표면화 회귀 수리 + 회귀 가드 케이스

장 / 단점 · 효율 · 품질 리스크

장점

  • 키 1개로 학습(1회 ~$0.32)·변환(~$0.04/곡) — GPU 분 단위(CPU 3.5h 대비)
  • 가중치 R2 영구 보관 = 보이스 1번 학습하면 계속 재사용
  • 동의 게이트 이중(api+워크플로) = 실존 타인 음성 차단 계약

단점·리스크

  • 실호출 미검증 — 트레이너 zip 레이아웃·비용 = 조사값(카나리아가 확정)
  • 보이스 소스·가중치 = R2 공개 URL(추측불가 키) — 비공개 필요 시 후속
  • RVC 커버 품질은 소스 녹음 품질에 민감(잡음 소스 = 음색 왜곡)

화면 실측 (헤드리스 E2E 10/10 · Uncaught 0)

학습 완료 — 파일 슬롯(ly 정본)·동의 토글·자동 선택
보이스 선택 = 촤르륵 팝업(selpop 계승)
완성곡 로드 → 보이스 입히기 노출
커버 완료 — 메타 = 보이스 커버 · 내 목소리

다음 1수 — 운영자 액션

  • 1) replicate.com 가입 → Account → API tokens 생성
  • 2) 레포 Settings → Secrets → REPLICATE_API_TOKEN 등록 (Billing에 카드 필요)
  • 3) "등록했어" 한마디 → 학습 카나리아 1건 실측 후 동작 확정 보고
