카드 프롬프트 (N회) 표식 — 반영본

첫 시도 = 기존 그대로(초록 진행). 자동 재시도 진입 시에만 (N회)를 덧붙임 — 기존 기틀(게이지·genstat 배너)에 표식만 추가. 픽토그램 없음. 재시도도 실패(=2회 이상) 시엔 기존 '실패' 표현.

A · 메뉴3 게이지바

'프롬프트' 라벨은 가운데 고정, 그 우측에 MUT '(1회)'. 진행 게이지(초록)는 그대로.

'프롬프트' 고정 + 우측 MUT (1)(숫자만·크기·정렬 라벨과 동일). 라벨은 안 밀림.

B · phase2(카드 상세) 진입부 배너

기존 .genstat 배너(텍스트+점, 픽토 없음)에 (N회)만 부착.

첫 시도 15분 초과 → 자동 1회 재시도. 기존 배너 '중' 뒤 띄어쓰기 + MUT (1회)(크기 동일).

반영: cardmake.sh 재시도 진입 시 status.json에 retry:1 기록·커밋 → build-viewer가 cards.retry 투영 → 게이지(.rn)·배너(.gen-rn)가 MUT '(N회)' 표식(크기=라벨/배너 문구와 동일). 색·애니 전부 기존 토큰(--mut·--accent·shine·pulse). 재시도도 실패하면 retry 해제되고 기존 '실패' 표현으로.
