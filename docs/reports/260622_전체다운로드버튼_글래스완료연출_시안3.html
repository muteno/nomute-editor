‘전체 다운로드’ 버튼 — 글래스 + 완료 연출 시안 3

현재 검정 솔리드(투박) → 프로스티드 글래스 베이스 공통. 저장 진행 중 저장 중… i/N 라벨, 완료 시 강조색이 버튼을 덮으며 ✓ 뜨고 다시 풀린다. 각 버튼을 탭하면 실제 저장 시퀀스(4장)처럼 재생돼. 썸네일 후보·카드 피드 전체버튼에 동일 적용 예정.

A 좌→우 강조색 와이프
진행률만큼 왼쪽부터 채워지고(다운로드 i/N과 1:1 매칭) 완료 시 꽉 차며 ✓ → 오른쪽으로 빠지며 풀림. 장점: 진행이 직관적·차분. 단점: 가장 점잖아 ‘액티브함’은 셋 중 보통.
탭 → 채움
B 중앙 발화 + 링 리플
중앙에서 강조색 원이 팡 터지며 버튼을 덮고 ✓ + 링이 바깥으로 퍼진다 → 다시 중앙으로 수축하며 풀림. 장점: 가장 ‘액티브’·완료감 큼. 단점: 진행률 표현은 라벨(i/N)에만 의존.
탭 → 발화
C 하단 게이지 → 풀필 플래시
하단 얇은 강조색 게이지가 i/N로 차오르다, 완료 순간 버튼 전체가 강조색으로 플래시되며 ✓ → 부드럽게 페이드 풀림. 장점: 진행 게이지+완료 플래시 둘 다(정보+임팩트). 단점: 요소가 가장 많음(게이지+플래시).
탭 → 게이지+플래시
