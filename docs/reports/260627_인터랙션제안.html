인터랙션 제안 — 최소화 눌림 · ▲→X 닫기

폰에서 직접 눌러/길게눌러 비교해봐. 번호로 골라주면 실제 뷰어에 박을게.

A. 최소화 버튼 눌림 (탭해봐)

최소화 버튼은 누르면 모달이 촤르륵 접혀 — 그 직전 “눌렸다” 피드백 3안.
M1 · 스냅 축소
이미지
콱 작아지며 흰 플래시. 가장 또렷·즉각적.
M2 · 초록 점등
이미지
살짝 줄며 accent 그린으로 점등. 접힘색과 연결.
M3 · 리플 + 축소
이미지
축소 + 초록 링이 퍼짐(물결). 가장 화려.

B. 복원 ▲ 길게눌러 X 닫기 (꾹 눌러봐)

▲ 길게(0.5s) → X로 모핑(겉둘레 없음). X를 정확히 탭=닫힘(액티브 후 사라짐) · 다른 곳 탭=취소(X→▲ 복귀). 3안은 X 가독 처리만 다름.
C1 · 강한 그림자
닫힘 ✕
꾹 → X / X탭=닫힘 / 밖탭=취소
X에 진한 그림자만(플레이트 없이 깔끔). 빨강 X.
C2 · 어두운 플레이트
닫힘 ✕
꾹 → X / X탭=닫힘 / 밖탭=취소
X 뒤에 라디얼 라벨식 어두운 판(가독↑·이미지 그 느낌).
C3 · 빨강 글로우
닫힘 ✕
꾹 → X / X탭=닫힘 / 밖탭=취소
X가 빨강 글로우로 떠오름(경고감 강조).
고르는 법 — 최소화 = M1/M2/M3 중 하나, 닫기 = C1/C2/C3 중 하나 (예: “M1 + C2”). 색·크기 등 더 손볼 것도 같이 말해줘.
※ 공통 동작(길게→X 모핑·X정확탭만 닫힘·밖탭취소·닫기 액티브)은 셋 다 동일. X 색(빨강/보라)·강조도 바꿀 수 있어.
