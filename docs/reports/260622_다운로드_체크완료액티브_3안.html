다운로드 클릭 = 체크 완료 액티브 — 3안

지금은 클릭하면 다운로드만 되고 끝 → 클릭 시 ✓ 완료 피드백 추가. 각 버튼을 여러 번 탭해서 비교하고, 마음에 드는 안(A/B/C)을 알려줘 → 모든 다운로드 버튼에 적용할게. (실제 다운로드는 그대로 동작)

A안 — 모프

↓ 아이콘이 ✓로 드로우-인 + 버튼이 초록으로 꽉 참. 0.9초 뒤 ↓로 복귀. (가장 명확·심플)

B안 — 팝 + 링

C안 — 게이지 채움

초록이 좌→우로 차오르고 ✓가 그려진 뒤 살짝 팝(settle). 전송 버튼(asksend)과 동일 계열. (가장 묵직·고급)
