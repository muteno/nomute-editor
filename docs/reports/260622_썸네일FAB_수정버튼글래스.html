썸네일 우측 빠른메뉴 FAB → '수정' 연필 버튼 외형 계승

첨부1의 우측 + 버튼(#rfab .rc)을 첨부2의 수정 연필 버튼(.rev-fab) 글래스 스타일로 통일.
아래 AFTER 버튼은 탭하면 눌림(scale)도 됨 — 실제 코드와 동일 값.

BEFORE — 기존

솔리드 밝은초록 채움(radial) + 어두운 아이콘. 글로우·반투명 없음.

AFTER — 수정 버튼 계승

반투명 글래스(--accent-dim) + blur + 초록 테두리(.42) + 초록 글로우 + 초록 아이콘. 수정 연필 버튼과 동일 외형.

세부(방사형) 버튼도 동일 글래스 통일

FAB를 열면 펼쳐지는 세부 버튼(전환·이미지·도구) — 기존 어두운 '곰발바닥' → 코어와 같은 글래스(초록 틴트·초록 아이콘·작은 글로우).

BEFORE — 세부
AFTER — 세부
↑ 참조 원본 = 뉴스 요약 '수정' 연필 버튼(.rev-fab)
