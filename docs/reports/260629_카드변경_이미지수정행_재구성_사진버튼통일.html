변경 · 검증

①"텍스트" 행에서 체크박스(.ed-match) 제거 → 라벨만.
②"이미지 수정 희망" → "이미지 수정" 라벨 변경.
③체크박스를 "이미지 수정" 라벨 행으로 이동 = 좌측 묶음(.ed-wishrow{justify-content:flex-start} + gap 10 = 옆 작은 간격).
④체크박스 라벨 → "텍스트 내용을 그대로 반영". (동작 동일: 체크 시 이미지가 텍스트 따라가고 희망 입력칸 잠김.)
⑤사진 첨부 버튼(#edAttach) 빈 픽토 = 흰색 → 초록(var(--accent)) = ly·카드뉴스 제작기 .file과 통일.
검증check_refs rc=0 · viewer JS 구문 OK · 토큰 게이트 baseline 이내. #edSync·#edWish 전부 getElementById 참조라 DOM 이동에도 sync 로직 불변. 단일 뷰어 순수 시각/구조 = 자가검증.

※ 체크박스는 "이미지 수정" 옆 좌측에, 텍스트영역 클립버튼(복사·붙여넣기·지우개)은 우측 — 폰 화면폭에서 안 겹침. 더 좁은 기기에서 빡빡하면 알려줘(간격/줄바꿈 조정).
