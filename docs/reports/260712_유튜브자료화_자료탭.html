유튜브 링크를 넣으면 전체 전사(무손실) + 별개 요약·핵심 논점·타임라인·주목 발언·팩트가 한 노트로 나오는 자료 탭. 정리 규칙이 레포 지침 파일(prompts/nb-make.md)이라 지시를 100% 따르고, 노트는 레포에 누적되며, 비용은 영상당 claude 구독 1콜(무료 축)이다.

장단점·효율성·품질 리스크

장점
  • 전사 패스스루 = "빠짐없이"가 프롬프트 약속이 아니라 기계 보장(평의회 만장 수렴 · 출력 예산 전부 분석 깊이로).
  • 기존 레일 재사용(자막기 STT·song 골격·폴오버 SSOT·rateGate) = 신규 표면적 최소.
  • 무료 축 고정 — 유료(GEMINI) 기능은 로드맵으로 분리(수동 슛 잠금 사상 유지).
단점·리스크(정직)
  • 유튜브 봇 인증 간헐(데이터센터 IP) — YT_COOKIES 시크릿 등록 시 안정(자막-온리 경로는 더 가벼워 실패율 낮음 = 추정).
  • 자동자막/STT 오인식은 고칠 수 없고 표시만 가능 — 배너 + (전사 기준) 딱지 + ⚠대조 플래그 3중 표시.
  • STT 폴백 1시간 캡(러너 CPU) · 초장편 LLM 입력 12만자 캡(절단은 coverage에 명시).
  • 라이브 E2E는 머지 후 실제 영상 1건 카나리아로 확정(이 보고서 시점 = 게이트·문법 검증까지).
효율성
  • 자막 경로 = 영상 다운로드 0 → 러너 수 분 · STT는 무자막 영상만 조건부 과금(Actions 분).
