검색 = 전면 블러 + 일치 결과 떠오름 (스포트라이트식)

돋보기 탭 → 뒤 화면 전체 블러, 타이핑하면 일치하는 결과만 블러 위로 떠오름(선명·그림자). 클리어존 위치 계산이 없어 PC·모바일 무관 · 결과는 블러 레이어 위 별도 층이라 타이핑 번쩍임도 없음. 프레임 안에서 직접 쳐봐(홈플·태평양·관저·반도체). PC/모바일 토글로 둘 다 확인.
