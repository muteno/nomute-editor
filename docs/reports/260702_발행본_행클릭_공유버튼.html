발행본 목록에서 행 아무 데나 누르면 그 공개 링크가 새 탭으로 열리고 동시에 클립보드에 자동 복사돼. 우측의 ‘복사’ 버튼은 ‘공유’ 버튼으로 바꿔서, 누르면 OS 공유 시트(카톡·메시지 등)가 뜨고 — 공유가 안 되는 환경이면 조용히 링크 복사로 대체돼. 픽토그램도 복사 아이콘 → 공유 노드 아이콘으로 바꿨고, 누름 모션(↗ 살짝 튀어오름)도 넣었어.

직접 만져보기 (실제 아이콘·모션·hover 재현)

행을 누르면 → 새 탭 열림 + 링크 복사 토스트 · 공유 버튼 = OS 공유(미지원 시 복사)

전 / 후 비교

전 (Before)

  • 행 클릭 = 아무 동작 없음 (버튼만 반응)
  • 우측 버튼 = 복사 아이콘 → 링크 클립보드 복사
  • 링크로 이동하려면 복사 → 새 탭에 붙여넣기 수동 2단계

후 (After)

  • 행 클릭 = 새 탭 열기 + 링크 자동 복사 (한 번에)
  • 우측 버튼 = 공유 아이콘 → navigator.share (카톡·메시지 등 OS 시트)
  • 공유 미지원/취소 시 = 링크 복사 + ✓ 플래시 폴백
  • 공유 버튼 누름 = ↗ 상승 모션(data-motion rise) · 행 hover 하이라이트

장 · 단점 / 효율 / 품질 리스크

장점

  • 열람이 1탭으로 — 발행 링크 확인이 빨라짐
  • ‘공유’가 폰 사용 흐름(카톡 전달)에 정확히 맞음
  • 복사도 덤으로 자동 — 링크 붙여넣기 필요할 때도 커버
  • 아이콘 SSOT(nm-svg.js)에 SHARE_SVG 등재 → 재사용 가능

단점·리스크

  • 행 전체가 눌리므로 오터치 우려 → 잠금·공유·삭제 버튼은 이벤트 격리(stopPropagation)로 제외
  • 만료 행은 클릭 비활성(go-gray) — 죽은 링크 안 열림
  • navigator.share는 데스크톱 일부 미지원 → 복사 폴백으로 안전
  • 새 탭이 포커스 뺏기 전 복사 먼저 실행 → 클립보드 실패 방지
