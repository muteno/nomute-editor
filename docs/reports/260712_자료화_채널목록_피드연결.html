후 (실물 스크린샷)

채널 URL 입력 → 영상 목록(제목·길이) · 행 [자료화] = 그 영상 노트 발사 · 우측 복사 = 전체 URL(개발자모드 추출 대체)
노트 최하단 「뉴스 요약 요청 — 피드로」 = 기존 요약 요청 파이프로 승격 → 피드 카드·카드뉴스·X·영상 기존 동선

요지

  • 채널 목록: 브라우저 스크롤링 불요 — yt-dlp flat-playlist가 채널·재생목록 전 영상을 목록으로만 초고속 추출(캡 200 · claude 0콜). 같은 입력칸(새 화면 0).
  • 피드 승격: 신규 파이프 0 — 노트를 조립해 기존 api/submit(요약 요청)로 제출. 다운스트림(피드·카드뉴스·X·영상) 전부 기존 그대로 = "가장 효용적으로 뭉침".
  • 수리(정직): 직전 #2103 병합 때 충돌 마커가 라우터·원장에 잔존한 채 main에 들어갔던 것 발견·제거(내용 무손실 검증). 재발 방지 = 커밋 전 마커 전수 grep 절차화.

변경 인덱스 — 12열 × 60행
