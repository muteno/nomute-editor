무엇이 바뀌었나 (쉬운 설명)

  • 위치: 하단/중앙/상단 3버튼 → 게이지바(0~100%). 왼쪽 끝=하단, 오른쪽 끝=상단. 이제 완성 영상에서 자막이 게이지 값 그대로 위아래로 실제 이동해 — 원본 영상 캡션·앱 UI에 가리면 조금만 올리면 됨.
  • 배경: 자막 뒤 검정 배경의 불투명도 게이지(0~100%) 신설. 0% = 배경 없음(종전 기본/깔끔 그대로), 올릴수록 진해짐. 어느 모양에서든 동작.
  • 예시칸: 크기(작게/보통/크게)·모양을 바꿔도 미리보기 칸 높이 58px 동일 — 출렁임 제거.
  • 미리보기 칸이 배경 게이지를 실시간 반영하고, 설정은 저장돼서(리번 포함) 다음에도 유지돼.

전 / 후

전 — 위치 = 3칩 고정(하단/중앙/상단) · 배경 조절 없음 · 예시칸 높이 크기 따라 출렁임

후 — 위치·배경 게이지 + % 표시 · 예시칸 배경 실시간(55%) · 높이 고정

크기 '크게'·카라오케 조합 — 예시칸 높이 동일(58px) 유지

합성 실측 (ffmpeg 실번인 프레임)

위치 게이지 0 / 24 / 55 / 100 — 픽셀 실측: 바닥 51px → 세이프존 429px(구 하단과 동일) → 정중앙 → 상단 155px. 전 구간 선형.

배경 게이지 0 / 44 / 90% — 박스 밝기 실측 85(없음)→46(이론 47.6)→8(이론 8.5) = 수식 그대로 반영.

장단점 · 효율 · 리스크 (5요소)

장점위치가 3단 고정 → 연속 무단(원본 캡션 회피 가능) · 배경 농도 취향대로 · 구 설정(localStorage)·구 캐시 뷰어·진행 중 작업 전부 하위호환 · 기본값 사용자는 종전과 시각 등가(하단 22%≈24%·6px 차)
단점중앙 정확값을 원하면 게이지를 55 부근으로 직접 맞춰야(스냅 없음 — 프리셋 누르면 24 복귀) · 배경은 줄 단위 각진 박스(ASS 한계 — 둥근 모서리 불가)
