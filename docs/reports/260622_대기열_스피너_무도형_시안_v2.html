셋 다 반영: ① 닫기 X = 글래스 버튼(실제 앱이 이미 표준 .tool-x라 네 이미지와 동일 — 시안만 옛 X였어서 교정) ② 재시도 횟수 1px 위로(중앙 보정·라이브) ③ 비활성 링크 상태 추가(원문 링크 없을 때 = 회색).

이번에 바뀐 것
• 우상단 닫기 X = 글래스 둥근사각(.tool-x) — 링크열(↗) 좌변과 좌측정렬 유지
• 재시도 숫자 2 = top:-1px로 살짝 위(스피너와 시각 중앙 맞춤)
• 비활성 ↗(회색) = 원문 URL 없는 전문 분석 행 — 누를 곳 없음 표시

상태 5종
• 처리중 = 앰버 스피너(도형X) + 앰버 ↗(원문) · 또는 회색 ↗(비활성) = 원문 링크 없음
• 재시도 N = 앰버 스피너 + 앰버 숫자(1px↑) — 숫자색 = 스피너색
• 완료 = 초록 한글 + 초록 ↗(뉴스요약) / 실패 = 빨강 한글 + 앰버 ↗(원문)
• 우상단 닫기 X = 글래스 .tool-x(네 이미지와 동일·이미 라이브)

숫자 1px가 과하거나 덜하면 말해 — 더 미세조정(0.5px·2px) 가능. 비활성 회색이 너무 흐리면 농도도 조절할게.
