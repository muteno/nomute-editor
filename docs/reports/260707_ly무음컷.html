자막기 무음 컷(컷 편집) 신설

260707 · 운영자 확정: 배치 = 자막 설정 칩 · 기준 = 발화(Whisper STT) · 기본 OFF · main 머지 #1814 + 라이브 카나리아 실측 완료

① 무엇이 생겼나

영상 첨부→자막 설정에서 컷 편집 켬→자막 생성→완성 영상 = 말 비는 구간 자동 컷 + 자막 싱크 자동 보정
  • 별도 분석 단계 없음 — 자막 생성이 이미 돌리는 Whisper STT의 발화 타임스탬프를 재사용(추가 LLM·과금 0).
  • 어절(word) 단위 타임스탬프 우선 — 문장 중간의 긴 침묵까지 컷 대상.
  • 컷 규칙: 발화 앞뒤 0.3초 보존 · 실제 갭 1.0초 미만은 유지(호흡 보존) · 머리/꼬리 무음 포함.
  • 컷하면 뒤 자막 시각이 전부 당겨지는데, 영상·오디오·자막이 같은 수식 하나로 이동해서 싱크가 구조적으로 안 어긋남.

② 전 → 후

축전후
무음 구간수동 편집 앱에서 직접 컷칩 하나로 자동 컷(완성 영상에 반영)
자막 싱크컷하면 SRT 다시 손봐야자동 재매핑(잘린 만큼 당김)
기본 동작—OFF = 종전과 바이트 동일 경로(회귀 0)
완성 노트—무음 3.6초 컷(1군데) 식으로 표기

④ 장단점·리스크

  • 장점: 원탭 컷 편집 · 자막 싱크 자동 · 추가 과금 0 · 기본 OFF라 기존 사용 무영향 · reburn(다시 입히기)으로 컷 켜고/끄고 재합성 가능(LLM 0).
  • 발화 기준의 성질: 말은 없지만 배경음(BGM·환경음)은 있는 구간도 잘림 — 운영자 선택 기준. 카나리아에서도 환경음 구간 3.6초가 잘렸다(의도 동작).
  • 품질 한계(정직): 컷 경계 정밀도 = 프레임 단위(약 21~33ms) · 미리보기 플레이어는 원본 타임라인(컷은 완성 영상에서 확인) · 컷 강도(0.3s/1.0s)는 고정값 — 튜닝 칩은 후속 후보.
  • 스코프: 컷은 자막 붙이기(완성 영상)의 하위 옵션 — 자막 없이 컷만 하는 모드는 이번 범위 밖(필요하면 말해줘, 쉽게 얹을 수 있음).

⑤ 후속 후보(도먼트)

  • 배경음악만 제거(운영자 질문): 가능 — Demucs/MDX 계열 보컬 분리 모델을 Whisper처럼 러너 로컬 실행(무료·키 불필요·7일 캐시). 켜면 완성 영상 오디오를 "목소리만"으로 교체. 비용 = 러너 시간 +1~3분/분·의존성 캐시 약 1GB. 품질 = 준수(약한 잔향 아티팩트 가능). 원하면 다음 작업으로 바로 구현.
  • 컷 강도 칩(약/중/강) · 미리보기에 컷 반영 · 컷-only 모드.
