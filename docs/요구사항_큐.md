> 1. 운영자 메시지 1개에 요구가 여럿이면 **그 자리서 원자 단위로 전부** 아래에 등록(⬜).

> 3. 완료(✅)는 **`git show origin/main`로 실측 확인한 것만**. 채팅으로 "했다" 금지.
> 4. 새 메시지 들어오면 먼저 여기 큐에 떼어 넣고 → 처리 → ✅.
