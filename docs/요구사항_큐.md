## ✅ 방금 완료 (이 대화)
- ✅ **카드 변경 모달 스크롤 버벅임(잼) 픽스**: `#editdlg`에 `backdrop-filter:blur(30px)`가 걸려 있어 자식 `.ed-body` 내부 스크롤마다 30px 블러를 **매 프레임 전면 재합성** → 큰 카드 이미지까지 같이 스크롤되며 심한 잼. bg가 84~93% 불투명이라 그 블러는 거의 안 보였음(고비용·저효용). → 블러 제거 + bg 불투명 .93/.96으로 글래스 유지(`::backdrop` 블러는 스크롤 경로 밖이라 유지). + `.ed-body`에 `-webkit-overflow-scrolling:touch`(iOS 관성)·`overscroll-behavior:contain`. baseline blur 92→90.
- ✅ **카드 변경 모달 칩 행 겹침(고스트 텍스트) 버그**: `.ed-body`(flex column·max-height)가 내용 초과 시 자식 `.ed-chips`(min-height:34px=명시 min이라 짜부 허용)를 34px로 압축 → 4줄 칩이 `overflow:visible`로 아래 "이미지 수정 희망" 라벨·textarea 위로 새던 것. → `.ed-body > * { flex-shrink:0 }`로 자식 비압축(대신 body 스크롤). 칩·이미지·textarea 다 자연 높이 유지.
- ✅ **PICK 버튼 = PASS처럼 플레인·글자만 형광**: `.sc-tg.pick`에서 초록 채움(`background:var(--accent-dim)`)·초록 테두리(`rgba(15,253,2,.4)`) 제거 → base `.sc-tg`(PASS) 상속 = 동일 다크 글래스. 글자색만 `var(--accent)`(형광 그린) 유지. 호버만 미세 형광 틴트. 폭(flex:1.5)은 1차 액션이라 유지.
- ✅ **게이지 진행 중 글자 점등 펄스**: `.prog .seg.cur .sl`을 정적 글로우 → `segCurGlow 1.5s ease-in-out infinite`(opacity 1↔.62·text-shadow .7↔.3) 펄스로. 게이지 차는 동안 위 글자(현재 단계)도 밝↔어두 반복. `prefers-reduced-motion`이면 정적 폴백.
- ✅ **첨부1 — 카드 합성 버튼 게이지바**: `makeCards`가 버튼을 *즉시* 상태박스로 교체해 게이지가 안 보였음(뚝딱) → `.mkbtn.firing::after` 하단 게이지바(pickGauge .72s) + makeCards가 **게이지 .72s 다 차고 나서** 교체(minGauge). 누르면 게이지바 좌→우.
- ✅ **첨부2 — 수집함 누적/신규 통일**: `.scrap-col.green`(누적)·`.scrap-col.black`(신규) 둘 다 **스포티파이 블랙 글래스**(동일 배경·흰 보더)로 통일 + 글로우 box-shadow(누적 .14 강·신규 .09).
