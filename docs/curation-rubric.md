# 큐레이션 라벨 기준 (운영자 취향 데이터 정본)

> 수집함(뷰어)에서 운영자가 누르는 **👎 · 별점 ★1~5 · PICK**이 `scraper/ratings.jsonl`에 누적된다(= 취향/학습 데이터). 각 라벨이 **무슨 판단을 뜻하는지**의 기준을 여기 못박는다. 나중에 이 데이터를 분석·학습에 쓸 때 라벨의 의미가 없으면 못 쓰므로. (운영자가 평가할 때의 일관성 기준이기도 함.)
>
> 정본 = 이 파일. 데이터 적재 = `.github/scripts/rate_record.py`(필드 `score`·`picked`·`dismissed`·`memo`·`action`·`reason`). 측정값(중요도 감쇠·burst·cross 등)과 별개인 **사람 판단 라벨**.
>
> **컬럼별 인터랙션 분리 (260616):** **누적** = 다매체 검증된 거 → ★1~5 + PICK(아래 §신호별 의미). **신규** = 걸러야 할 인박스 → **[👎][PASS][PICK] 3택 + 사유(객관식) 무조건 선택**(아래 §신규 트리아지). 사유 = 주관식 부담↓·나중 정량화↑ 지표. DB 필드 `action`(down/pass/pick)·`reason`(사유 키).

## 신규 트리아지 — action + reason (정량화 지표)
신규 컬럼에서 각 액션을 고르면 **사유를 무조건 객관식으로 택1.** 그 사유가 "왜 그렇게 판단했나"의 정량 지표(나중 알고리즘이 "이건 이런 의미로 제꼈구나" 설명 가능).

### ⏭ PASS — 👎(관심없음) 통합 · 3분류 + 점수(0~2) (260621 운영자 개편)
버튼 = `PASS / PICK / ✓` 3개(👎 thumb-down 폐지 → PASS로 통합). PASS = 제외(`dismissed`·부정 취향신호).
**점수(0~2)는 reason명에 내재 = 기록만, 아직 랭킹 알고리즘 미반영(나중에 먹임).** 작을수록 "더 가치 없음".
- **스팸 (0점)** — 기사로서의 가치가 가장 없으며, 애초에 '수집 필요가 없는 기사'. *(옛 down '가치없음' 흡수.)*
- **홍보성 (1점)** — 지방·지자체·기업·주가 띄우기용·산하 행정단체의 홍보성 보도자료 = 수집 필요 거의 없음. *(옛 down '국지적'+'기업·정책' 흡수.)*
- **가치없음 (2점)** — 기사가 화제성은 있지만, 콘텐츠화 하기엔 다수의 유의미한 반응을 이끌어내기 어려운 기사. *(옛 pass '늦은보도'+'애매함' 흡수.)*
- **직접입력** — 위에 없는 사유(주관식 한 줄).

### ✅ pick (큐레이션 발사)
- **속보/사건** — 빠르게 전달하면 모두가 감정적 격동을 느낄 수 있을 것이라 판단.
- **토픽/논란** — 범국민적인 토픽·주제라고 판단.
- **대세픽** — 국민적 인기. 지금 다수가 주목하는 대세 흐름(트렌드·화제성)이라 판단.
- **지식** — 효과적인 콘텐츠를 만들면 팔로워가 유용하다고 느낄 것이라 판단.

## 신호별 의미 (누적 컬럼 ★·PICK)

### 👎 (관심 없음 / dismissed) — *신규 기사에만 해당*
- **이슈로서의 가치가 없음.** 콘텐츠화 할 경우, 팔로워가 관심을 가지지 않고, 매거진의 가치에 의구심을 느낄 수 있음.
- 동작: 리스트에서 숨김(DB엔 `dismissed:true` 잔존).

### ★1 (별점 1개)
- 콘텐츠화 할 경우, 팔로워가 관심을 가지지 않고, 매거진의 가치에 의구심을 느낄 수 있음.

### ★2 (별점 2개)
- 콘텐츠화 할 경우, 팔로워가 관심을 가지지 **않을 것이라 확신**하는 주제. 반복 시 매거진의 가치에 의구심을 느낄 수 있음.

### ★3 (별점 3개)
- 콘텐츠화 할 경우, **무작위의 관심**을 가질 수도 있을 것이라 추측하는 주제.

### ★4 (별점 4개)
- 콘텐츠화 할 경우, **팔로워의 30% 이상**이 관심을 가질 것이라 판단하는 주제.

### ★5 (별점 5개)
- 콘텐츠화 할 경우, **팔로워의 50% 이상**이 관심을 가질 것이라 판단하는 주제.

### PICK (큐레이션 발사) — 선택한 주제는 아래 중 하나
1. **[팔로워 맞춤]** 시의적절하며 / 콘텐츠화 할 경우, 팔로워의 50% 이상이 관심을 가질 것이라 판단하는 주제.
2. **[속보]** 국민 대다수가 초기에 접하게 되면 감정적 격동을 (일으키는) 격정적 사건(스캔들, 정치 사건 등).
3. **[전국적 사건·재해]** 국민 대다수가 이 주제에 대해 갈라지지 않고 범인간적인 같은 공감을 하는 주제(연쇄살인, 국제범죄, 재난, 추모 등).
4. **[핫토픽]** 전국민이 공감대를 느껴본 주제(정치 사건, 기업 이슈, 노조 등).
5. **[유용한 정보 또는 경제/시사/상식]** 이 정보를 잘 전달하면 팔로워들이 단순 가십성이 아닌, 삶을 영위하는 데 지속적인 도움을 줄 수 있다고 판단한 내용.

## 메모(memo)
- ★1·★5에서 주관식 한 줄(왜 그 점수인지). DB `memo` 필드. 라벨에 근거를 붙이는 정성 데이터.
