# 기틀 복제팩 v2 — 간결 강령 골격 (타 레포 전달 정본)

> **용도**: 운영자의 *일하는 방식*을 다른 레포에 심는 단일 전달 파일. 새 레포 첫 세션에 이 파일 하나를 첨부하고 아래 1.의 지시문을 복붙하면 끝.
> **v2 = 간결 강령(15칙) 골격.** 구 v1(법령체 0~6부 템플릿)은 `_versions/260714_1005_공유본갱신/` 동결 — 길어서 지시가 파묻히는 문제로 운영자가 폐지(260714). 현행 정본 = nomute-editor 레포 최상위 CLAUDE.md.
> 디자인 시스템이 있는 레포면 `docs/디자인방식론_복제팩.md`(방식론)·`docs/디자인토큰_제1핵심명령_복제전문.md`(값)도 같이 전달.

## 5. (선택) 기계 게이트 — 강령을 지키게 만드는 장치

> 강령은 소프트 룰이다. 실제로 지켜지게 하는 건 커밋 전 기계 게이트 — 새 레포에 게이트가 없으면 아래 최소 스켈레톤을 `shared/check_refs.py`로 생성해 시작하고, 레포가 커지면 게이트를 누적한다. (이미 더 강한 게이트가 있으면 그걸 유지 — 다운그레이드 금지.)

```python
#!/usr/bin/env python3
"""check_refs 최소 스켈레톤 — CLAUDE.md가 언급한 레포 경로 실존 검사."""
import re, sys, pathlib
ROOT = pathlib.Path(__file__).resolve().parent.parent
def main():
    txt = (ROOT/"CLAUDE.md").read_text(encoding="utf-8")
    pats = re.findall(r"`([\w./가-힣_-]+/[\w./가-힣_-]+)`", txt)
    bad = []
    for p in sorted(set(pats)):
        if any(ch in p for ch in "{}*<>"): continue
        if not (ROOT/p).exists(): bad.append(p)
    if bad:
        print("❌ CLAUDE.md가 가리키는 경로 미실존:"); [print(" -", b) for b in bad]
        return 1
    print("✅ check_refs 통과 — 경로 참조 실존."); return 0
if __name__ == "__main__": sys.exit(main())
```
