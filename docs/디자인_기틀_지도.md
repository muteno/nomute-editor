## 5. "어디부터 봐?" — 진입점 4개

- **값**이 궁금 → `viewer/index.html :root`
- **규칙**이 궁금 → `CLAUDE.md [4]` + SessionStart 훅 주입(`.claude/hooks/design_digest.py`) + `nomute-design` 스킬 (구 §🎨 전문 = 260701 압축 해체 → 포인터 정정 260717)
- **컴포넌트** 만들 때 → `docs/CII_컴포넌트계승인덱스.md`
- **눈으로** 보려면 → `구성도/index.html`(허브)

## 6. ⚠️ 색 = 뷰어별 독립 (구조 토큰만 공유)

핵심 오해 방지(260628 확립 — 원문 = 구 CLAUDE.md §🎨 STAGE3·현행 강제 = CLAUDE.md [4] 팔레트 예외 조항):
- **색은 뷰어마다 독립** — `index`(뉴스)=터쿼이즈 브랜드(`--accent:#00EED2`), `thumb`/`ly`/`k`/`comp`=차분한 warm 툴톤(자기 `:root`). **색 공유 안 함**(재색칠 회귀 영구 금지).
- **공유되는 건 구조 토큰만** — radius·spacing·blur·z·버튼·타이포·모션 = `viewer/tokens.css`(4뷰어 `<link>`).
- 그래서 "교과서=단일 팔레트 파일" 기대는 어긋남 — **값 정본은 `index :root`(자동거울 `base.css`)** 하나지만, 뷰어 색은 각자.

## 7. 브랜드 팔레트 = 폐쇄형 (260704 확립)

역할색은 유일. 새 색 창작·유사색 추가 금지(운영자 확인):
- 🩵 코어 `--accent #00EED2`(터쿼이즈) · 🟡 `--accent-2 #d8ff3d`(이력/최소화·라임) · 🔴 `--accent-3 #e23b2a`(danger·강보수) · 🟡 `--accent-4 #FFE13D`(warn·arm·thumb·골드레몬) · 🩵 `--accent-5 #0cd0f7`(info·naver·약진보)
- 의미색 = accent 스케일 **별칭**(`--danger`=accent-3, `--warn`/`--amber`/`--arm`=accent-4, `--info`=accent-5)
- 편향 5단(강진보 `#3a6ddb` ↔ 중립 흰 ↔ 강보수 accent-3)
- 중간 회색 = `--mut` 단일(`--fg-2` **폐지** — 보조텍스트도 mut)
- 확립본 = `docs/reports/260704_브랜드컬러_확립본.html`
