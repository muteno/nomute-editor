NoMute · Utility

클립보드 다운로더 가이드

URL 복사하고 Ctrl+Shift+D만 누르면 downloader2.bat이 그 URL을 받아 다운로드하는 시스템. 이제 그냥 실행(더블클릭)해도 클립보드의 URL을 자동으로 물어온다. (v5.2 · 클립보드 자동 + 연속 입력)

URL 복사 → Ctrl + Shift + D → 다운로더 실행 + URL 자동입력

1 한눈에 보기

어딘가에서 영상·링크 URL을 복사한 뒤 단축키를 누르면(또는 bat을 그냥 실행하면), 그 URL이 downloader2.bat에 자동으로 들어가 다운로드가 시작돼. 받은 파일은 로컬 + 구글드라이브(내 드라이브\Shared) 양쪽에 저장돼.

v5.2에서 바뀐 점 ① 어떻게 실행하든(단축키·더블클릭) 클립보드에 URL이 복사돼 있으면 자동으로 첫 입력으로 들어감 — 별도 러너(.ps1) 불필요.
② 클라우드 저장이 OneDrive → 구글드라이브(내 드라이브\Shared)로 변경.
연속 입력(q로 종료)은 v5.1 그대로.

2 설치 파일 + 최초 설치 · PC당 한 번 · PowerShell 안 씀

필요한 파일 두 개가 이 문서 안에 내장돼 있어. 아래 버튼으로 받으면 끝 — 매번 새로 받을 필요 없고, 이 HTML 하나만 있으면 어느 PC든 재설치 가능.

  • ① downloader2.bat → OneDrive의 고정 위치에 넣는다(기존 파일 덮어쓰기):
    …OneDrive\황세웅\6. Nomute\창고\05. Utility\downloader2.bat
  • ② 단축키설치.bat → 아무 데나 받아서 더블클릭 → [OK] 설치 완료 뜨면 끝. 이 파일은 지워도 됨.
옛 방식(.ps1) 쓰던 PC면? 그냥 위 ①② 그대로 하면 돼. 설치기가 같은 단축키를 쓰던 옛 바로가기를 자동으로 해제하고 새 방식(bat 직접 실행)으로 바꿔줘. 러너 ps1을 매번 새로 받을 필요는 원래도 없었고, 이제는 아예 안 씀 — %USERPROFILE%\nomute 폴더는 지워도 됨.
실행할 때 Windows 경고가 뜨면 인터넷에서 받은 bat이라 SmartScreen이 막을 수 있어 → "추가 정보" → "실행". 브라우저가 다운로드를 막으면 "유지"를 눌러.

3 평소 사용법 · 두 가지 방법, 둘 다 연속 입력

방법 A — 단축키 (빠름)

  • 다운로드할 URL을 복사 Ctrl+C
  • Ctrl+Shift+D 를 누른다 → 검은 창이 뜨고 복사한 URL이 첫 입력으로 자동 처리됨
  • 다운로드 후 그 창에서 계속 — 다음 URL을 복사해 붙여넣거나(우클릭) 직접 입력
  • 다 끝나면 q 입력해 종료

방법 B — 더블클릭

  • downloader2.bat을 더블클릭 — 이때도 클립보드에 URL이 있으면 자동으로 첫 입력돼(v5.2). 없으면 평소처럼 URL 붙여넣기 → q로 종료
클립보드가 비었거나 URL이 아니면 자동 입력 없이 그냥 URL 입력 대기로 시작해(오작동 방지 — http(s):// 로 시작할 때만 자동 사용). 창에 직접 붙여넣으면 됨.

4 자막

영상과 함께 자막(.srt) + 편집용 텍스트(.txt)를 자동으로 받아. 자막 txt는 번호·타임코드·중복 줄을 제거한 깔끔한 본문이라 바로 쓰기 좋아.

  • YouTube — 가장 안정적. 수동 자막 + 자동생성 자막 둘 다 시도.
  • X · 인스타 · 틱톡 · 페북 · 스레드 — 플랫폼에 자막 트랙이 거의 없어서 영상만 받아지는 게 정상.
자막이 안 생기면 ffmpeg부터 확인 srt 변환은 ffmpeg.exe가 해. yt-dlp 폴더(…\05. Utility\yt-dlp\)에 ffmpeg.exe가 없으면 자막이 .vtt로만 남거나 안 생기고, txt 변환도 실패해. 유튜브인데 자막이 안 나오면 99% 이 문제.

자막 언어를 바꾸려면 BAT 위쪽 set "SUBLANG=en,ko" 줄 수정 (예: ko 만 / en 만). 유튜브 자동자막이 가끔 누락되면 en.*,ko.* 로 넓힐 수도 있어.

5 저장 위치

구분경로
임시(로컬)%USERPROFILE%\Downloads\yt-dlp
최종(클라우드)%USERPROFILE%\Google Drive 스트리밍\내 드라이브\Shared

로컬에 먼저 받고 → 구글드라이브 Shared 폴더로 복사하는 이중 저장이야. 사용자 계정명(Hwang 등)이 PC마다 달라도 %USERPROFILE%이 알아서 맞춰줘. 구글드라이브가 없는 PC면 [알림] 띄우고 로컬에만 저장해(가짜 폴더 안 만듦).

6 다른 PC로 옮길 때

downloader2.bat은 OneDrive 고정 위치에 있으니 동기화만 되면 자동으로 따라와. 새 PC에서 할 일은 하나뿐:

  • 이 HTML에서 단축키설치.bat을 받아 더블클릭 1회 — 끝.
경로는 자동으로 맞음 bat 위치는 %OneDriveCommercial%, 구글드라이브는 %USERPROFILE% 기준이라 사용자 계정명(Hwang 등)이 달라도 그대로 작동해. 회사 OneDrive 로그인 + 구글드라이브 데스크톱 로그인만 돼 있으면 됨.

⚠ 구글드라이브 데스크톱이 없으면 다운로드는 되지만 로컬에만 저장돼(실행 시 [알림] 표시).

항목위치보관

정리: 남겨야 하는 건 이 HTML + OneDrive의 downloader2.bat 두 개뿐. 나머지는 전부 재생성 가능.

8 문제 해결

증상원인해결
받은 bat 실행하니 Windows 경고가 뜸 인터넷에서 받은 파일이라 SmartScreen이 확인 요청 "추가 정보" → "실행" (한 번만)
Ctrl+Shift+D 눌러도 반응 없음 단축키 미적용, 또는 다른 앱과 충돌(브라우저의 "모든 탭 북마크"가 같은 키) 로그아웃→재로그인 또는 재부팅 · 그래도 안 되면 단축키 변경(아래 9번)
유튜브인데 자막이 안 생김 yt-dlp 폴더에 ffmpeg.exe 없음 ffmpeg.exe를 yt-dlp 폴더에 넣기
SNS 영상 자막이 없음 플랫폼에 자막 트랙 자체가 없음(정상) 영상만 받으면 됨 — 코드 문제 아님
실행했는데 클립보드 URL 자동입력이 안 됨 클립보드 내용이 http(s)로 시작하지 않음 (URL 앞에 다른 글자) URL만 다시 복사 후 재실행 — 또는 창에 직접 붙여넣기
[알림] 구글드라이브 폴더 없음 — 로컬에만 저장 구글드라이브 데스크톱 미설치/미로그인 구글드라이브 설치·로그인 후 재실행 (그 전에도 로컬 저장은 정상)

9 설정 바꾸는 법

단축키 변경

단축키설치.bat 메모장으로 열어 위쪽 HOTKEY 줄 수정 후, 다시 더블클릭.

set "HOTKEY=Ctrl+Alt+J"   :: 원하는 키로

저장 경로 변경

downloader2.bat에서 GDRIVE·CLOUD 줄 수정.

set "GDRIVE=%USERPROFILE%\Google Drive 스트리밍\내 드라이브"
set "CLOUD=%GDRIVE%\Shared"

자막 언어 변경

downloader2.bat에서 SUBLANG 줄 수정.

set "SUBLANG=en,ko"   :: ko / en / all / en.*,ko.* 등
