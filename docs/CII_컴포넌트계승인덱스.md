# 📑 CII — 컴포넌트 계승 인덱스 (Component Inheritance Index)

> **운영자 정본 · 260628 · 분신술 10인 도출.** UI/UX 컴포넌트를 **표로 인덱싱**해 "항상 계승"시키는 기틀. §🎨 "📦 정본 컴포넌트 인벤토리"(불릿)의 **기계가독 표 승격판**. 새 버튼·입력칸·모달·아이콘을 만들거나 이식할 때 **반드시 이 표의 정본 셀렉터를 계승**(재설계 금지). 충돌 시 §🎨·viewer `:root`가 정본.

## 왜 (재발 방지)
드리프트 2대 근원: ① thumb/ly/k/comp가 index `:root` 토큰 미상속(raw 값 박힘) ② 같은 컴포넌트 4중 복제(한 곳만 고쳐짐). 이 표 = "어떤 컴포넌트는 *무엇을* 계승해야 하는가"의 단일 출처. 코드가 표와 어긋나면 드리프트.

## 위계 (값·규칙·강제 3층 — §🎨 그대로)
1. **값 SSOT** = `viewer/index.html :root` 토큰 (색·`--btn*`·`--r*`·`--sp*`·`--blur*`·`--z*`·타이포·모션).
2. **규칙·인덱스** = 이 표 + §🎨 + `구성도/00_가이드북_버튼인터랙션.html`.
3. **강제** = `shared/check_refs.py check_design()`(raw 값 baseline·5파일 커버) + `build_design_mirror.py`(거울 정합).

## 표기 규칙
- **크기·마진·반지름·z = 토큰만**(raw px 금지). 없으면 가장 가까운 토큰, 정 없으면 `:root`에 토큰부터 추가.
- **아이콘 = SVG만**(이모지·`✕`문자 금지). 같은 의미는 같은 path.
- **닫기/최소화/버튼묶음 = 헤더 우측**(`margin-left:auto`는 *묶음 래퍼*에 — 개별 버튼에 걸면 갇혀 좌측붙음 = editdlg 버그).
- **이미지 위 오버레이 버튼 = 불투명 `var(--bg)`**(아이콘색만 accent · hover/active 배경 불변 = 누름 scale+아이콘 모션 · 이미지 위 `.sbtn` 액션[투표·수정·재생성]도 동일 플레이트 = 한 행 혼재 금지 · 구 검정 20% 반투명 폐지 = 가시성 우선 · 운영자 260709). 일반 다운로드 = `.dlbtn`(accent 10%). ⚠️ **카드 위 다운로드류(저장 `.save`·오버레이 PNG `.ovl` 등)는 빠짐없이 `.dlbtn`** = 클릭 액티브 통일(안 붙이면 누락 · 260629 `.ovl` 교정).
- **누름 :active = press *토큰 티어* 계승** — viewer :root `--press-pico/xs/s/m/l` 5토큰 사다리(아래 「:active 눌림 효과 토큰화(260628)」 항목 = 현행 · 티어 용도는 :root 주석 정본)를 `scale(var(--press-*,폴백))`로 참조. 게이지·보라·카드는 셀렉터 화이트리스트 제외. raw scale 창작·고정 0.96 금지(옛 리터럴 표기 폐기 · 260702).
- **AI 브리프 강조 2층(정본 · 전 브리프 표면 공용 · 운영자 260714 "모두 통일")**: 마크다운 `*문구*` = **1층 = 강조색 볼드**(`.tbrief-t b` = `--accent`·`--fw-x` 800 · 제일 중요·수치/전환점 · 섹션당 0~1개) / `**문구**` = **2층 = 강조색 없이 볼드만**(`.tbrief-t b.tb-em` = 본문색 `--fg`·`--fw-b` 700 · 어느정도 중요·핵심 명사/동사구 · 섹션당 1~3개). 위계 = 강조색800 > 볼드700 > 본문400 3단. 파서 = **더블스타(`**`)를 싱글(`*`) 앞에 배치**(같은 위치서 먼저 매칭 · `renderSnsTrends`·`renderChan` 공용 관용구) · 생성기 프롬프트도 2층 규칙 동봉. ⚠️ **경계 — 뉴스 요약 본문(`.cem`)은 이 규칙 비대상(통일 금지)**: 거긴 `*문구*`·`**문구**` **둘 다 문구만 강조색 `<i>`(별표는 화면 미표시)** + 복사 시 `codeCopyText`가 `data-mk`로 **별표를 복원**해 카드뉴스 합성기 입력 포맷을 보존하는 *별개 축*이다(정본 = index `.cem` 규칙·codeCopyText). 여기에 브리프 2층(`**`→본문색 볼드)을 이식하면 ① 색 의미 붕괴(강조색→본문색) ② 복사 포맷 파손(합성기가 못 읽음) = **양쪽 다 깨짐**. "브리프와 같은 별표니 통일하자"는 오판 차단용 경계(운영자 260714). 새 브리프/요약 표면 = 이 2층 문법 재설계 금지·계승(단 `.cem`류 합성기 입력축은 위 별개 규칙).
- **소머리(구분자) `•` = 텍스트 흰색(`--fg`)·`--fw-x`(800) / 블릿 형광(`--accent`)·`--fw-b`(700) · `--fs-label`(13px)** — 크기·굵기 *토큰*(리터럴 금지). 특수(이전제작·최소화)만 블릿 이력색(`--hist-accent` — 260705부터 라임옐로=accent-2) · 텍스트는 *항상* 흰색. **회색(`--mut`) 소머리·블릿 없는 소머리 금지.** 블릿 메커니즘 = **블록은 `::before content:'• '`(공백·뉴스 정본 동일), flex(`.csec`)는 `content:'•'`+`gap:6px`**(둘 다 6px급 간격·layout별). 정본 = 뉴스 `index .cref-lbl`. `.gospec`(명세 readout)은 소머리 아님(예외). ⚠️ **flex `.csec`에 토글(`.ovfmt`/`.onoff`) 붙으면 토글 세로패딩(3px·탭영역)이 행을 키워 `•`가 ~3px 내려앉음 → `.csec .ovfmt, .csec .onoff { margin-block:-3px }`로 행높이 기여 상쇄**(탭영역 보존·토글 유무 무관 첫 블릿 화면선 통일·운영자 260629 저작권탭 교정). 강제 = `check_refs.check_soremeori()`.
- **특이점(변곡·이벤트) 주석 = 한 번 꺾이는 점선 리더 정본(운영자 260715 "특이점은 항상 저런 식으로 표기")**: 차트 위 특이점 표기는 **전높이 세로선 금지** → **데이터점 → 수직 → 팔꿈치 1회 → 수평 → 라벨** 점선 리더로 그 일자 데이터점을 *정조준*. 톤 = 주석 크롬(`stroke="var(--mut)"`·`stroke-dasharray="3 3"`·`opacity .6`·`linecap/linejoin:round` — 강조색 곡선과 위계 분리) · 라벨 = 팔꿈치 수평 끝(우측단 `x > W-mR-70` = 좌향 꺾임 `dir=-1` — 차트 밖 삐침 방지) · 라벨 선반 = 항상 점 위(`shelfY = min(mT+5, py-8)`). 호버 = 그 일자 값(추이)+요약(`note`) 툴팁(`.ch-tip.ev`) + 흰 호버점(`.ch-dot`). 정본 = index `chChart` events 렌더(SVG path 1개 생성 · `insta_events.json` 등 특이점 데이터 공용 = *단일 렌더러라 드리프트 원천 차단*). 새 차트·특이점 표면 = 이 리더 문법 재설계 금지·계승.
