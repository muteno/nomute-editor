## details.so vault · navigation-01 「Navigation Dropdown」 (수집 260723)

- **출처**: https://www.details.so/vault/navigation-01 (유료 'Get Code' 상품의 공개 프리뷰)
- **수집 방식**: 본 페이지 = Astro 셸이라 본문 없음 → 프리뷰 iframe `https://www.details.so/vault-previews/pv-be0cc28c52c30dba/index.html` 직접 수집(28.9KB 자립 HTML). CSS·마크업 = 공개 서빙 원문 그대로(부록 A). **동작 JS = javascript-obfuscator 난독화(유료 보호) → 미해제** — 아래 동작 기술 = CSS 상태 클래스·마크업·데모 관찰 기반 재구성.
- **한 줄 정의**: 고정 헤더 중앙 필 내비 + **패널 3종(메가/심플/피처드)을 단일 플라이아웃 컨테이너가 크기 모프하며 갈아끼우는** 드롭다운 + 스크롤 시 헤더가 글래스 캡슐로 수축 + 모바일 clip-path 시트·아코디언.

### 2) 핵심 디테일 12 (왜 좋은가 + 값)

1. **단일 플라이아웃 크기 모프**: 메뉴마다 드롭다운을 따로 띄우지 않고 컨테이너 하나가 `width/height/transform`을 `.5s var(--ease-dropdown)`으로 모프 → "패널이 살아서 늘어나는" 연속감. 배경판(`__bg`)과 클립판(`__viewport`)을 분리해 배경 페이드(.18s)와 크기 모프를 따로 재생.
2. **expo-out을 CSS `linear()`로 구현**: `--ease-dropdown: linear(0,.005,.02 2.1%,.081 4.7%,.467 15.7%,…,1)` — 초반 15.7% 시간에 46.7% 도달 후 긴 정착. spring 라이브러리 없이 순 CSS.
3. **방향 인지 패널 전환**: 이웃 메뉴로 이동 시 나가는 패널 `translate(∓200px)`+opacity, 들어오는 패널 반대편 200px에서 진입(`.is-leaving-left/right`·`.is-enter-left/right`) — 이동 *방향*이 애니에 반영돼 공간감 유지. opacity는 .18s로 크기 모프(.5s)보다 빨리 끊어 잔상 방지.
4. **첫 오픈 = `.is-instant`**: 첫 등장은 transition을 속성 나열만 남기고 duration 제거(무애니 점프) → 이후 메뉴 전환부터 모프. "0에서 늘어나는" 어색한 첫 프레임 봉쇄.
5. **닫힘 대기 상태 = −4px·scale .98**: 닫힘 시 `--flyout-y:-4px; --flyout-scale:.98` + `visibility` 전환을 `0s linear .18s` 지연 → 퇴장 페이드가 다 보인 뒤 숨김. 등장·퇴장 진폭이 4px/2%로 미세 = "은은한 퇴장" 사상(nomute-design §6과 동일 축).
6. **호버 브리지(::before 12px)**: 내비와 플라이아웃 사이 10px 틈을 `::before{top:-12px; height:12px}` 투명띠로 이어 포인터가 틈에 빠져도 안 닫힘 — 메가메뉴 조기닫힘 고질병의 정석 해법.
7. **이동 필 하이라이트 2벌**: 상단 `nav-hover-bg`(필 radius)·드롭다운 내부 `nav-dropdown__pill`(radius 13 · bg `#7a7a7a27`)이 활성 항목 rect로 `transform/width(height) .2s var(--ease-out-cubic)` 이동 — 항목별 개별 hover 배경 대신 **한 장이 미끄러짐**. 첫 표시는 `.is-instant`.
8. **형제 디밍 = `:has()`**: 한 항목이 hover/focus/expanded면 나머지 상단 항목 opacity .55, 활성만 1(.15s) — JS 0줄로 포커스 유도.
9. **스크롤 수축 헤더**: 스크롤 시 `header.is-scrolled` → shell max-width 1024→720(`transition:max-width .8s var(--ease-dropdown)`) + 글래스 캡슐(`.header-bar` radius 28 · `#000000b8`+blur(10px)+그림자) opacity 0→1(.35s). 상단 = 투명 내비, 스크롤 = 떠 있는 좁은 캡슐.
10. **프리뷰 이미지 정착 줌**: 메가 패널 우측 이미지 = 전환 시 opacity .2s + `scale(1.1)→1`을 `.6s var(--ease-out-cubic)`로 정착(미세 켄번즈). 피처드 카드 hover도 `1.06→1`(.5s) — "확대"가 아니라 "정착" 방향이라 차분함.
11. **모바일 시트 = clip-path·아코디언 = grid-rows**: ≤900px에서 내비가 헤더 아래 시트로 — `clip-path: inset(0 0 100% round 24px) → inset(0 round 24px)`(.55s)라 **모서리 라운드를 유지한 채** 열림. 서브메뉴 = `grid-template-rows: 0fr→1fr`(.45s)로 height:auto 애니 해결. 햄버거 2줄(16×1.5 · ±3.25px)→X 회전 .35s. `overscroll-behavior:contain`·`max-height:calc(100dvh − 110px)`.
12. **이미지 로드 페이드 + reduced-motion**: `data-image-fade` 스크립트(부록 A 하단)가 로드 완료 후 500ms `cubic-bezier(.22,1,.36,1)` 페이드인, `prefers-reduced-motion`이면 즉시 표시 — 이미지 팝인 방지.

## 모프 타이머 필 「Set timer → 분 휠 → 카운트다운 링」 (수집 260723)

- **출처**: 운영자 첨부 영상 2건(동일본 · LoveThreads.Net 워터마크 재공유 클립 · 원 디자이너 미상 컨셉 데모) — 15s · 1092×718 · h264.
- **수집 방식**: 코드 원문 없음 → ffmpeg 1fps 프레임 15장 실측 + 픽셀 색 샘플링 = **관찰 재구성**(아래 값 = 프레임 실측 근사). 구현 정본 = `구성도/타이머 모프.html`(.tm · 작동 데모 + 정지 프레임 + 토큰 규약 표).
- **한 줄 정의**: 「타이머 설정」 필 버튼 → 클릭하면 **컨테이너 하나가 크기·radius를 모프**해 분(min) 휠 피커 카드 → 시작하면 **테두리 자체가 빨간 진행 링인 카운트다운 필**(일시정지|시간|취소) → 취소·종료 시 필로 역모프.

### 1) 구조 해부 (상태 3 + 모프)

```
.container (단일 · width/height/radius 모프 · 내용 3겹 크로스페이드)
├ [idle]  필 416×140 r≈70 — 스톱워치 픽토(회색 원판 위 흰 라인) + "Set timer"
├ [pick]  카드 566×476 r≈44 — 좌측 세로 휠(1~60 순환 · 중앙 선명·대형, 이웃 blur/페이드 사다리)
│         + "mins" 라벨(중앙 우측) + 우하단 초록 "Start" 필
└ [run]   필 648×296 r≈64 — 테두리 인셋 진행 링(sw≈5 · 빨강) + Pause(주황)|M:SS(대형 tabular)|Cancel(회색 필)
```

- 진행 링 = 잔여 시간에 비례해 스트로크가 줄고, **갭 중심 = 상단 중앙 고정**(양방향 대칭으로 벌어짐).
- 휠 = 60↔1 순환 스크롤 · 중앙 행만 선명·확대, 거리 1·2·3행은 opacity·blur 사다리(원근감).
- 모든 버튼 = 클릭 시 딤 액티브(필 전체가 어두워졌다 복귀).

### 2) 핵심 디테일 7 (왜 좋은가)

1. **단일 컨테이너 모프**: 상태마다 다른 팝업을 띄우지 않고 한 그릇이 늘었다 줄었다 — vault navigation-01의 배경판 모프와 동축(연속감 = 같은 물체라는 서사).
2. **테두리 = 진행률**: 링을 안에 따로 그리지 않고 컨테이너 윤곽 자체가 게이지 — 정보가 크롬을 안 늘리고 실린다.
3. **갭 상단 중앙 대칭**: 소모가 시계방향 한쪽이 아니라 12시에서 양쪽으로 — 잔여가 항상 좌우 대칭이라 한눈에 읽힘.
4. **휠 원근 사다리**: 중앙 선명·대형 + 이웃 blur/페이드 = 선택값이 광학적으로 유일 초점.
5. **의미색 3축**: 시작=초록 · 잔여=빨강 · 일시정지=주황 — 상태 의미가 색으로 즉독.
6. **좌우 대칭 액션 배치**: Pause|time|Cancel — 파괴(취소)와 보류(일시정지)가 시간을 사이에 두고 등거리.
7. **역모프 원복**: 종료·취소가 새 화면 전환이 아니라 처음 필로 되감김 — 시작점 복귀가 공간적으로 명시.
