## A. 진행바 ⇄ NMT 통합 (메인)
- **A1** ⬜ 대상 = 카드 하단 4단계 진행바(`progGauge`/`.prog` — 실패·요약·프롬프트·카드). 현재 라벨 안 보임·뭉침. (`viewer/index.html` CSS L199–219, JS L527–546)
- **A2** ⬜ 2026급 UIUX로 교체(리서치 반영·100/100 미려).
- **A3** ⬜ 라벨 가독(문자열 또렷이 보임).
- **A4** ⬜ 안 뭉치게(여백·간격 충분).
- **A5** ⬜ **NMT 하이퍼링크 nav를 진행바에 통합**(둘 따로 X → 진행바 스텝이 곧 점프 버튼). 별도 `.nmt` 레일 제거.
  - 점프 대상(NoMuTe): **시작**(모달 top) · **N**=인스타 본문(요약) · **M**=카드뉴스 프롬프트 · **T**=카드 이미지(모달 팝업에 리로드된 생성 이미지).
  - 매핑(진행 상태 ⇄ 점프): 요약=N(IG) · 프롬프트=M(text_done) · 카드=T(done). [해석 — 사용자 확인 가능]
- **A6** ⬜ **하이퍼링크 정밀화**(현재 "대충 언저리" = 개짜증 포인트). 각 섹션의 **바로 위 구분선**을 기준점으로, 그 구분선이 **팝업 스크롤 최상단(스티키 헤더 바로 아래)**에 오도록 정확히 점프. (스크롤 컨테이너=`<dialog id=dlg>`, offset=`.dlg-h` 높이 보정.)

## B. 날짜 — IN vs WRITE
- **B1** ⬜ 두 날짜 구분 표기: `IN 6/14(일) 11:00` + `WRITE 6/11(목) 11:00`.
  - IN = 큐 적재시각 = `a.file` 앞 YYMMDD-HHMM 파싱. WRITE = `a.date`+`a.time`(보도시각).
- **B2** ⬜ **강조는 WRITE**(IN 아님).
- **B3** ⬜ IN은 **오늘 일자일 때만** 강조.

## F. 모달 섹션 헤더 = 번호 부제 + 이모지 (3 점프 섹션 통일)
- **F1** ⬜ 요약 phase 헤더: **📄→✍️** 교체, 텍스트 **"1. 기사 요약 · 시사점"**. 부제 스타일(현재보다 키우되 상단 기사 타이틀보단 약간 작게). 대상 = `#ph-sum > summary`(현 "📄 기사 요약 · 시사점"). 옆 지침버전 칩(gvChip) 유지.
- **F2** ⬜ 카드 phase 헤더: **"2. 카드뉴스 — 텍스트·프롬프트"**(가운데점 · 포함), 글씨 키움(부제). 대상 = `cardSec` sumlabel(`text_done`). gvChip 유지.
- **F3** ⬜ 3번째 점프 대상(카드 이미지) 헤더도 **"3. …"** 부제 형태로 통일(NMT의 T 도달점).
- → **3 섹션 번호 부제**: `1.기사 요약·시사점(✍️)` / `2.카드뉴스—텍스트·프롬프트` / `3.카드 이미지`. A5 NMT가 이 셋으로 점프, A6 정밀 앵커.
