## 1. 기틀 (깨면 안 되는 것)

1) **전사 = 패스스루 (제1 기틀 · 평의회 앵글1·3·7 만장 수렴)**
- a) 전체 전사는 `yt-dlp` 자막(`nb_sub.py` 파싱) 또는 Whisper STT(`ly_stt.py`) 산출을 **기계 삽입** — claude가 전사를 재출력·교정·요약하지 않는다.
- b) 왜: LLM 재기입 = 원문 오염("빠짐없이" 위반·인용 근거 붕괴) + 출력 토큰 낭비. 패스스루 = 무손실 기계 보장 + 출력 예산 전부를 분석 깊이에 투자.
- c) 미래 세션 주의: *효율화* 명목으로 claude에게 전사 재출력시키는 개조 금지(§기틀 보호).

2) **claude 1콜 = 분석 섹션만** — `prompts/nb-make.md`(지침 SSOT · `NB_GV` 해시 도장) → JSON {kind·one·summary·points·chapters·quotes·facts·terms·topics·entities·use·coverage}. 모델 = `shared/model_env.sh` PIPE_MODEL(opus·생성 축 §모델 d) · 폴오버 = `shared/claude_transient.sh` SSOT(4계정 · check_refs 강제).

3) **품질 가드(앵글7 조건부 통과의 조건 = 전부 이행)**
- a) 신뢰도 배너: `tr_src`(subs=업로더 자막/subs-auto=자동/stt=Whisper)를 노트·뷰어에 상시 표기, 자동·STT면 오인식 경고.
- b) 인용 = 전사 자구 + 타임스탬프 + **기계 대조**(nbmake.sh가 정규화 부분일치 검증 → `v:false` = 뷰어 ⚠ 표시).
- c) 화자 실명 추정 금지(명시 식별만) · 수치 `(전사 기준)` 딱지 · `[추론]` 분리 · 스폰서 격리 · 커버리지 자기점검 · `TRANSCRIPT_FAILED` 정직 실패 게이트 · 인젝션 차단.

4) **노트 흐름 = 역피라미드 (뷰어 렌더 순서)**: 메타(제목·채널·길이·원본 링크·유형 칩·전사 출처 배너) → 한 줄 → 요약(300~700자 독립 완결) → 핵심 논점(주장+근거+반론·타임스탬프 원자화) → 타임라인 → 주목 발언(자구 인용) → 팩트·수치(접기) → 인물·용어(접기) → 활용 포인트 → **전체 전사(맨 아래 기본 접힘 + 자수 배지 + 전문 복사 + .txt 다운로드)**.

5) **타임코드 = `[mm:ss]`(1h+ = `h:mm:ss`) + 유튜브 `&t=` 딥링크** — 전사가 타임코드를 유지해야 쇼츠 클리퍼(`prompts/clip-make.md`) 등 다운스트림이 물릴 수 있다(앵글6).

## 2. 파이프 (song-make 미러 + ly STT 레일)

1) 흐름: 뷰어 `nb.html` 폼(URL+선택 지시) → `functions/api/nb.js`(rateGate·KST id) → `nb-make.yml` 디스패치 → {메타(`--dump-single-json`) → 자막 2패스[수동→자동 · 언어 = 원어>ko>en · **영상 다운로드 없음**] → 없으면 오디오만 받아 Whisper(ly 레일·runner-setup 캐시)} → `nbmake.sh`(claude 1콜) → `viewer/nb_out/<id>/note.json` 커밋 → Pages 배포 → 뷰어 폴(error.log 우선·SPA-HTML 가드·ts 신선도).
2) 목록 = `viewer/nb_out/index.json`(빌드 산출 · `build-viewer.mjs` song_out 미러 · 캡 500 · 검색 표면 필드만) → 뷰어 우상단 목록 팝업(msgpop 계승).
3) LLM = **영상당 1콜**(구독 OAuth · 과금 0). 제미나이·R2 미사용(텍스트 산출 = git 커밋으로 충분).

## 3. 캡·한계 (정직)

1) 자막 경로 상한 = `NB_MAX_SEC`(기본 4시간) · STT 폴백 상한 = `NB_STT_MAX_SEC`(기본 1시간 — 러너 CPU whisper 소요) · 라이브 방송·재생목록 = 거절.
2) 전사 LLM 입력 캡 = `NB_LLM_MAX`(기본 12만 자) — 초과분은 분석 미포함을 프롬프트에 명시하고 coverage에 기록(전사 자체는 전량 보존).
3) 유튜브 봇 인증(데이터센터 IP) 간헐 발생 가능 → `YT_COOKIES` 시크릿 등록 시 안정(ly와 공유 · 자막-온리 경로는 다운로드보다 가벼워 실패율 낮음 = 추정).
4) 화면(슬라이드·자료 화면)은 못 봄 — 자막/음성 텍스트 기반(NotebookLM 동일 한계).
5) 비동기(발사 → 자막 경로 수 분·STT 수십 분) — 실시간 챗 아님(구독 OAuth = Actions 경유만 · §파이프라인 7-c).
6) 완료 푸시 없음(v1 = 인모달 폴 + pend 재개) — 필요 시 §완료알림 배포 게이트 패턴으로 후속.
