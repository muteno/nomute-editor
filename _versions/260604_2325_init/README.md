# nomute-editor — 노뮤트 카드뉴스 에디터 (클라우드/폰용)

뉴스 기사를 노뮤트 시사 카드뉴스(텍스트 + AI 일러스트)로 만드는 시스템.
**Claude Code에서 이 레포를 열면 `CLAUDE.md`가 자동 로드돼 노뮤트 뉴스 에디터로 작동한다.**

## 폰/클라우드에서 쓰는 법
1. Claude Code(폰 앱 또는 claude.ai)에서 **이 레포(`muteno/nomute-editor`)를 연다.**
2. **기사 텍스트(또는 이슈 이미지·키워드)를 붙여넣는다.** → 자동으로 0단계·앵글·IG·Thread·썸네일·시사점까지.
3. **`4`** → 카드뉴스(카드별 🍌 Nano / ✳️ GPT 프롬프트) → 🚦STOP.
4. **`ㄱ`** → Drive 저장 + 자동화(이미지+합성) → **15분 뒤** 전 카드 합성본 + 피드백 질문이 이 채팅에 뜸.

- 셸 = **bash**(리눅스 클라우드 샌드박스 — curl·date·python3).
- 드라이브 발사 = **Zapier 커넥터**(폰 계정에 연결돼 있어야 함).
- 앞에 `,` 붙여도 됨(로컬 트리거 습관) — 이 레포에선 이미 에디터라 무시하고 진행.

## 파일
- `CLAUDE.md` — 마스터(자동 로드): 3단계·화풍·악센트·자물쇠·기틀 보호
- `01_지침_*` — 본체 로직(시스템프롬프트 역할)
- `02_라이브러리_*` — 카메라/조명/디스패치 코드 (4-B 온디맨드)
- `03_자동화_*` — 파이프라인·과금·Zapier·프리뷰(bash)
- `04_구조_*` — 전체 지도 / `05_리뷰_*` — 피드백·취향 코퍼스
- `PROJECT_MEMORY.md` — 브랜드룰·결정 / `_산출/_베스트/` — 취향 코퍼스

## 주의 (기틀 보호)
- **합성은 항상**(파이프라인 기본). 시안 토글 안 씀.
- **기틀(분리·3단계·참조방식·INVARIANTS) 변경은 반드시 사용자 확인 후.** 모델 맘대로 금지 — 갱신 중 누락에 의한 품질 열화(뫼비우스) 방지.
