## 파이프라인 (확정 — Apps Script v5 `code.gs` 소스로 검증)
```
지침(에디터) → 카드 MD(🍌만) → Drive Prompt/{yymmdd_hhmmss_주제}/동명.md

```

## parsePrompts 포맷 (★ 엄수 — 이게 진짜 운영 포맷)
카드 헤더 `### [카드 N]` … `**이미지 프롬프트**` + ` ```text ` 블록. (`**텍스트**` 블록 = 카드 문구, 합성용.)
- 운영 정규식: `/###\s*\[카드\s*(\d+)\][\s\S]*?\*\*이미지\s*프롬프트\*\*\s*\n+([\s\S]*?)(?=\n###\s*\[카드|\n##\s|$)/g`
- ` ```text … ``` ` 펜스는 자동 제거. **MD엔 🍌 Nano만(✳️ GPT 제외).**
- ⚠️ 로컬의 `parser_v2.py`/`[장면]`/`Scene core` 변형은 부차 파이프라인 — 운영 아님. 헷갈리지 말 것.

## 합성 = 항상 (에디터 로직엔 토글 없음)
**Drive에 저장하면 무조건 합성까지 된다.** 에디터는 합성 여부를 결정하지 않고, MD에 합성을 끄는 플래그를 넣지 않으며, 카드 텍스트를 항상 채운다.
(※ 시스템 사실 — 디버그용 참고만: Apps Script는 `- 시안 모드: true`이거나 모든 `**텍스트**`가 비면 합성을 건너뛴다. **단 에디터 로직은 이걸 절대 사용하지 않는다.** 순수 이미지만 뽑을 일은 운영자가 콘솔/수동으로 직접 처리.)
- 완료 마커 `.gen_complete`(fail=0 = 전 카드 성공). MD 24h+ 미수정이면 스킵.

## 과금 (Google Cloud — Drive 저장 = 유료 발사)
- **Gemini API 💰**(이미지 생성) + **Cloud Run 💰**(합성; 시안 모드면 미발생).
- ⚠️ 그래서 지침이 4-B 후 🚦STOP을 강제 → GO('ㄱ') 받고만 4-C 저장. 텍스트 미확정 저장 금지.
- `GEMINI_API_KEY` · `GDRIVE_SA_JSON`(Drive 접근 서비스계정 키) = **운영자 보유, 여기 미보관(민감).**

## Zapier 발사 (Claude Code에서 직접 — 챗앱보다 강한 부분)
1. `google_drive_create_folder` (parent = Prompt 폴더 ID) → 폴더 ID 획득
2. `google_drive_create_file_from_text` (folder=그 ID, title=`{yymmdd_hhmmss}_{주제}.md`, convert=false, file=MD)
→ Apps Script가 1분 내 잡아 이미지 생성. 생성 후 `retrieve_files`로 장면NN.png + .gen_complete 확인.
- KST 타임스탬프 (클라우드=리눅스 bash): `TZ='Asia/Seoul' date '+%y%m%d_%H%M%S'`

## GPT Image 2 (✳️) — 수동, API 미연결
- ✳️ GPT 프롬프트는 **자동화에 안 들어간다.** API 키 미연결 상태. 운영자가 ChatGPT의 **GPT Image 2에 수동 복붙**해 생성.
- 그래서 ✳️는 **채팅 출력 전용**, Drive MD에 미저장(자동화 무손상). 🍌(Nanobanana 2 Pro)만 Drive→자동화.

## 프리뷰 — 이미지를 이 채팅에 직접 띄우기 (실측 확정 260604 · 한 방에)
운영자가 드라이브 안 가고 이 창에서 `_final_`을 보는 법. **base64 풀다운 금지**(토큰 폭증). **썸네일 URL을 한 콜로 다 받아 bash(curl)로 다운:**
1. **전 카드 주소 한 방에** — `google_drive_make_api_get_request`(Zapier, **1콜뿐**): url=`https://www.googleapis.com/drive/v3/files`, querystring `q`=`'{폴더ID}' in parents and trashed=false`, `fields`=`files(name,thumbnailLink)` → 그 폴더 전 파일의 thumbnailLink를 한 번에. *(⚠️ a24c2baa `get_file_metadata`는 thumbnailLink 안 줌 → 이 cheap 경로는 Zapier만. 실측.)*
2. 응답에서 `_final_*.jpg`만 골라, 각 thumbnailLink 끝 `=s220` → **`=s2048`** (거의 원본; s1024=105KB·s2048≈원본 317KB).
3. bash `curl -sL "{url}" -o {temp.jpg}` 로 각각 **디스크 직행**(컨텍스트·Zapier·GCP 0). 클라우드=리눅스 샌드박스라 curl·date·python3 다 됨.
4. `Read {temp .jpg}` → 채팅 렌더 (전 카드 한꺼번에).
**비용/속도**: GCP **0** / **Zapier = 1콜(2task), 카드 수 무관**(한 방에라서) / 1장 다운 ~1-2s. 픽셀완벽 최종검수만 base64. 완전 Zapier·GCP 0 원하면 로컬 동기화(Drive for Desktop).
