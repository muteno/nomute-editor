7 케이스 — 요약 모달 헤더

단순 액션 + 상태형 체크를 한 글래스 알약에 통합한 정본 예. 체크 탭→ON, 요약 탭→다운로드.

요약 원문 ↗

체크 ON = 알약 E3(테두리+은은한 채움) + 강한 글로우 링 + 켤 때 펄스 · 요약 탭 = 초록 플래시 + 다운로드 아이콘 ↓바운스 · 원문 탭 = 흰빛 펄스 + ↗ 너지 · 체크 = 원형(알약 곡률 패밀리) · aria-label·:has 포커스 링·reduced-motion 가드.

9 전체 다운로드 시퀀스 버튼

‘전체 다운로드’ — 썸네일 후보·카드뉴스·썸네일 생성기 대기열 공용. 글래스(형제 .sbtn 톤) + 좌→우 강조색 채움(진행 i/N·검정 글자 드러남) + ✓ 팝 → 즉시 복귀(드레인 없음). 탭하면 재생.

대기 "전체" → 진행 "저장 중… i/N"·좌→우 채움(검정 글자 드러남) → 완료 ✓ 커졌다 작아지는 팝 + 글로우 → 즉시 복귀(드레인 없음·글자 안 가려짐) → 원복. 베이스 글래스 = 형제 .sbtn 톤. 구현 = .cref-dlall(index)·.jsave(thumb) + 상태 .dlseq-busy/done·복귀 .dlseq-snap · 진행 --p(0~1) · JS dlAllSeq/jSaveSeq · 기틀북 §9(.k-dlseq). §5 전송 게이지의 사촌.

8 접근성 · reduced-motion
