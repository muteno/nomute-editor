## 3. 인터랙션 언어 — 3겹 피드백

1. **눌림(press)** — `:active{transform:scale(.9);filter:brightness(.88)}` (알약=`.94`·아이콘=`.85~.9`). 누르는 동안. ⚠️ 실전 이식은 raw scale 금지 — viewer :root의 `--press-pico/xs/s/m/l` **5토큰 사다리**를 `scale(var(--press-*,폴백))`로 계승(260628 토큰화 · 이 예제 수치는 데모 렌더값 · 260702).
2. **시맨틱 모션** — 누르면 아이콘이 의미대로 움직임: **회전**(기본·새로고침 360°·닫기) / **↓바운스**(다운로드·저장). `document` 위임 1핸들러(capture)가 `closest()`로 아이콘버튼 잡아 svg에 `ic-spin`/`ic-bounce` 부여(`e.defaultPrevented` 가드). 기존 svg 애니(nav·refresh hover·FAB)는 셀렉터서 제외.
3. **탭 피드백** — 완료=초록 플래시(다운로드 ✓) / 열림=흰빛 펄스(외부 링크). 의미별 색 분리.

```
@keyframes ic-spin   { to { transform:rotate(360deg); } }
@keyframes ic-bounce { 0%{transform:translateY(0)} 45%{transform:translateY(2.5px)} 100%{transform:translateY(0)} }
.ic-spin{animation:ic-spin .5s var(--ease)} .ic-bounce{animation:ic-bounce .42s ease}
```

## 4. 재확인 — 비용/파괴 액션만

돈/시간 나가거나 되돌리기 어려운 액션엔 게이트. 공짜·즉시·가역엔 **안 건다**(피로 방지).

| 액션 | 등급 | 재확인 장치 |
|---|---|---|
| PICK(수집함) | 비용·일회 | 2탭 arm→게이지→**소진 잠금**(원복 X·탭=피드) |
| 카드 슛(유료 이미지) | 비용·비가역 | PIN 입력 게이트 *(현재 즉발 — 운영자 260614 정책)* |
| 슬롯 재생성·수정요청 | 비용 | 2탭(`.confirm` arm→발사) |
| 카드 변경·내역 삭제 | 파괴 | 확인 팝업 |
| 모달 내 send(요약요청·재분석) | 비용 | 모달 게이트(내용 필수)로 충분 — 2탭 불요 |
| 요약/다운로드/원문/복사/토글 | 단순·가역 | 없음(즉시 ✓) |

**전송 상태머신 라벨(전 메뉴 고정)**: 대기("전송"/"발사"/"합성"/"생성") → **"재확인"**(arm 앰버·3s 타임아웃) → **"전송 중…"**(초록 게이지 .72s) → **"완료"**+✓ → 원복. 실패=shake+빨강. (대기 라벨만 메뉴별 동사 허용.)

## 7. 케이스 — 요약 모달 헤더 (정본 예)

단순 액션 + 상태형 체크를 한 글래스 알약(`.dlbox`)에 통합.

- **체크 ON** = 알약 E3(테두리+은은한 채움) + 강한 글로우 링 `0 0 16px rgba(accent,.30),0 0 0 1px .25` + 켤 때 펄스(`dlrip`).
- **요약 탭** = 초록 플래시(`okflash`) + 다운로드 아이콘 ↓바운스. **원문 탭** = 흰빛 펄스(`openflash`) + ↗ 너지.
- 체크 = **원형**(알약 곡률 패밀리) · 네이티브 input 숨김+형제 박스 · `dlFull.checked`→전체 다운로드 계약 보존.
- `downloadSummaryHtml`은 `.dl-lbl` span만 갱신(아이콘 보존).

## 9. 전체 다운로드 시퀀스 버튼 (`.k-dlseq` · 정본 패턴)

‘전체 다운로드’ = 여러 이미지를 순차 저장하는 버튼(썸네일 후보·카드뉴스·썸네일 생성기 대기열 **공용**). §5 전송 게이지의 사촌(완료 후엔 드레인 대신 즉시 스냅 복귀).

- **베이스** = 프로스티드 글래스 = **형제 아이콘버튼(`.sbtn`·검색 버튼) 톤 그대로**(`rgba(8,15,11,.3)` + 흰 테두리 `.14` + `blur(--blur-s)`) = 앱 통일. (구 어두운 불투명/강조 테두리 폐지.)
- **상태 라벨**: "전체" → **"저장 중… i/N"**(좌→우 네온레몬 `--accent-4` 채움 `::before` scaleX=`--p`(0~1)·**채움 위 글자는 `--on-amber`**라 차오르며 드러남 · 색 = 전체 버튼 네온레몬 통일 운영자 260706) → **"완료" + ✓**(커졌다 작아지는 팝 `k-dlpop` 0.3→1.3→1 + 글로우) → **즉시 복귀**(`.dlseq-snap` = transition none·드레인 게이지 없음 → 글자 가려짐 0) → 원복.
- **진행률** = `--p` 인라인 변수(다운로드 i/N과 1:1). **아이콘** = 다운(`.dn`)↔체크(`.chk`) 스왑 한 벌(`.dlseq-ic` grid 겹침).
- **구현**: `.cref-dlall`(index·thumb **통이식 단일 셀렉터**, JS `dlAllSeq` — 구 thumb `.jsave`/`jSaveSeq`는 260705 통이식으로 폐기) + 상태클래스 `.dlseq-busy/.dlseq-done`·복귀 `.dlseq-snap`. 라이브 앱은 미니멀 변형(채움 없음·글자 네온레몬 유지·받음 `.dl-done`=`--mut` 260706). 기틀북 CSS = `.k-dlseq.is-busy/is-done`·`.is-snap`.
- reduced-motion = 채움/팝 즉시 전환(`transition·animation:none`).
