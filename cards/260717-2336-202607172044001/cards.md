# 🕯 새벽 비명 신고 5시간 뒤…의정부 일가족 4명 숨졌다

**[프롬프트 설계]**
- 화풍: A 한국 웹툰 수채화 — 참변·미성년 피해·추모 정조라 극화의 날선 고발보다 수채의 먹먹함이 이 사건의 침묵에 맞다.
- 분위기: 새벽의 시린 정적에서 아침의 무거운 여운으로. 분노보다 침묵으로 멈추는 절제된 톤(#추모·미성년 = 강도 상한 낮춤).
- 연출 방향: 독자훅 = 여덟 살·열두 살 두 아이의 나이 앞에서 멈추는 먹먹함 + 새벽 비명과 낮 참변 사이 다섯 시간의 역전. 무엇을 = 닫힌 문 앞에서 멈춘 초동대응과 그 사이 흘러간 시간이라는 물음(💡 시사점). 사람을 직접 그리지 않고 부재·흔적·닫힌 회색 현관문에 의도를 쏟는다(#자살자해·#아동피해 = 간접 시각화·추락 장면 비묘사). 사건의 강조점 = '문' — 끝내 열리지 못한 경계. thumb_dispatch(LGT02 차가운 새벽광·SG-09 부재)의 조명 톤·정조만 상속, 앵글은 카드마다 분산.
- 독자 동선: 起 카드1 → 承 카드2~3 → 轉 카드4 → 結 카드5 · 훅=카드1 끝(다섯 시간 전 예고·단서형)+카드2 끝(비명 신고→카드3 회수) · 감정 피크=카드4(문 하나 사이·두 아이 나이) · 착지=비명 앞에서 사회가 어디까지 문을 두드릴 수 있었나.
- 연속성 앵커: 반복 인물 (없음) — 인물 부재 중심 구성. 반복 장소 = A plain gray steel apartment front door in a dim corridor(카드3·5 Scene에 동일 문자열 복사).
