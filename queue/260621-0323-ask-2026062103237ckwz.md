# '삼전닉스' 반도체 계약학과, 서울대 자연계 넘었다…의대도 턱밑

## 🧷 한줄 요약

- 대학별: 한양대 반도체공학과(SK하이닉스) 98.0 / 고려대 반도체공학과(SK하이닉스) 97.0 / 성균관대 반도체시스템공학과(삼성전자) 96.0 / 서강대 시스템반도체공학과(SK하이닉스)·연세대 시스템반도체공학과(삼성전자) 각 95.0점.
- 의대 비교: 지방권 의대 평균 97.2점(고려대 반도체공학과와 유사, 한양대보다 낮음) / 경인권 의대 99.0점 / 서울권 의대 98.8점.
- 기업별: SK하이닉스 계약학과 3곳 평균 96.7점 vs 삼성전자 2곳 평균 95.5점 — SK하이닉스가 1.2점 높음.
- 임성호 종로학원 대표: "반도체 계약학과와 의대 동시 합격 시 선택에 따라 양쪽 합격선이 모두 영향", 서울대 공대 대신 계약학과를 택하는 사례도 늘 수 있다고 전망.
