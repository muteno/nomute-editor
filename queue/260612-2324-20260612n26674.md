# 선관위 잇단 '개표 입력 오류'…전북·경기 교육감 선거서 표 뒤바뀌어

## 🧷 한줄 요약
2026년 6·3 지방선거 개표에서 선관위가 투표소 명칭을 잘못 기재·입력해 전북·경기 교육감 선거 결과가 실제와 다르게 집계됐고, 사후 검증까지 줄줄이 뚫리며 선거관리 신뢰가 도마에 올랐다.

## 📰 Fact (확인된 사실)
- 전북교육감 선거: 전주시 완산구 중화산1동 **제3투표소** 개표 결과가 같은 동 **제1투표소** 결과로 잘못 입력됨. 투표록에 투표소 명칭이 오기된 게 발단. **1,104표가 사실상 잘못 처리(증발)**됐다.
- 투표함 표기 오류 → 개표 입력 → 사후 검증까지 **최소 세 차례** 오류를 걸러낼 기회를 놓침.
- 당락에는 영향 없음: 서거석 당선인이 남승희 후보를 **약 11만8,644표** 차로 앞섰음.
- 경기교육감 선거에서도 결과 입력 오류가 추가로 확인 → 선관위 **2연속 대국민 사과**.
- (별개·같은 맥락) 중앙선관위는 투표용지 인쇄 오류(4매 2,000장 폐기)를 발표, 야당(이준석 의원 등)이 부정선거 의혹을 제기하며 국회 정개특위에서 논의. 255개 지역선관위도 유사 사례 검증 중.
- 출처: YTN(뉴스라인) / 교차 확인: 뉴데일리·한국일보·전민일보·전북도선관위 사과문·네이트뉴스.

## 🔎 Inference (추론·해석)
- "당락에 영향 없음"은 사실이나, 핵심 쟁점은 결과가 아니라 **검증 시스템이 3중으로 뚫렸다는 점**이다 `[확신 높음]`.
- 전북에 이어 경기까지 같은 유형 오류가 나온 건 개인 실수보다 **개표·입력·검증 절차의 구조적 허점**을 시사한다 `[확신 중간]`.
- 인쇄 오류·입력 오류가 잇따라 보도되며, 선거 불신 여론(부정선거 의혹 포함)을 키울 소재가 됨 `[확신 중간]`.
- 단, 현재까지 확인된 건 '의도적 조작'이 아니라 '관리 부실'이다 — 부정선거 단정은 사실 근거 부족 `[확신 높음]`.

## 🧭 다각도 해석
- **이렇게 볼 수도**:
  - (불신 강조) 반복되는 오류 자체가 선거 신뢰의 근간을 흔든다 — 결과가 안 바뀌었어도 "걸러질 거란 믿음"이 깨진 게 더 큰 손실.
  - (실용·관리) 수개표·전산 이중검증 강화, 투표록 표기 표준화 같은 절차 보완이 핵심이지 음모론은 본질을 흐린다.
  - (정치) 부정선거 프레임으로 확전되면 진짜 고쳐야 할 시스템 개혁이 정쟁에 묻힐 위험.
