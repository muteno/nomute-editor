## 1. ⚠️ 기존 클링(/k) 레인은 보존한다 — "대체"의 정확한 뜻

- **클링은 안 날린다**(운영자 260714): 200만원 상당 구독이 무료로 확보돼 있고, 프롬프팅만 잘하면 품질이 잘 나옴. **단 클링은 API/MCP가 없어 자동화 불가** → 수동 복붙 레인으로 유지.
- "기존 영상 프롬프팅 메뉴를 대체"의 뜻 = **영상 탭의 중심 플로우가 스토리보드 파이프라인으로 재편**되는 것이지, 기존 /k 프롬프트 빌더(kling·veo·seedance 문법 산출) 엔진 삭제가 아님.
- 결과 = **촬영 2레인**이 스토리보드 하류에서 분기:

```
감독 선택(오퍼스/페이블) → 화제·이야기 입력
        → [설계도] 마스터시트(V2, Element 락용) + 콘티(storyboard-v1 · 12컷)

             A. 클링 레인(수동·무료) — 기존 /k 엔진으로 Kling 복붙 프롬프트 번들 산출
             B. 시댄스 레인(자동·크레딧) — seedance-continuity-builder → 힉스필드 MCP
                generate_video(seedance_2_0) · 생성 직전 get_cost 프리플라이트 = 필수 게이트
```
