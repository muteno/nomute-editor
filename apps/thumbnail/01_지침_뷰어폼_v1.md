> **정본 코드 = `viewer/thumb.html`.** 이 문서 = 그 폼의 **구조·관례 SSOT**. 렌더링(PIL 스크립트)은 `00_지침_썸네일과오버레이`·`MEMORY`가 정본이고, **여긴 입력 UI/UX(탭·goSpec·슬롯·수정복원·자간안내)만** 다룬다.
> ⚠️ **왜 만들었나**: 이 폼이 매번 흔들린 건 *정해둔 게 없어서*다(운영자 260624). 이제 여기 박는다 — 새 세션·압축 후 썸네일 폼 작업이면 **이 문서부터** 읽고, 충돌 시 이 문서가 우선(코드 실측으로 갱신).

## §0. 대원칙 (제일 중요 — 이걸로 안 흔들림)

1. **흰 = 고정 → 백엔드 불변.** 폼/goSpec/슬롯 작업은 **`functions/api/thumb.js`·`.github/workflows/thumb-make.yml`·`apps/thumbnail/nomute_*.py` 안 건드린다.** 헤더는 항상 흰칸을 렌더하고, 배경(nobg) 등은 *프론트 옵션*일 뿐. → **폼 UI는 자유롭게 바꿔도 렌더 회귀 위험 0.** (이게 그동안 빠른 반복이 안전했던 이유.)
2. **LLM 토큰 0.** /1~/4 = 순수 PIL 렌더. `effort`·`모델` 레버 **해당없음**. (유일한 외부호출 = 맞춤법 자동교정 `functions/api/spellcheck.js` = 비-LLM 프록시·무료·비차단.)
3. **디자인 = §🎨 토큰 계승.** raw hex/blur/accent 창작 금지 → `var()` 토큰. 새 버튼/컴포넌트는 기존 인벤토리 계승(재설계 금지). 다운로드=`.dlbtn`.
4. **검증 = `check_refs` + node 파싱(자가검증).** 폼 변경은 §✅ⓒ 국소 뷰어 손질 = 자가검증으로 끊는다(동작 로직·`:root` 토큰·파이프라인 끼면 더 엄격). `python3 shared/check_refs.py` + 인라인 `<script>` `new Function()` 파싱.
5. **반영 = §수정 모드.** 백업 → 수정 → check_refs → PR → main 머지 → `git show origin/main` read-back. 머지 후 Cloudflare 1~2분(뷰어 `_headers` no-cache = 하드새로고침 불요).

## §3. goSpec = 결과 명세 = 컨트롤 (핵심 패턴 · 기틀)

> **정의**: 제작 버튼 *위* 한 줄. 결과물 명세를 보여주면서 **동시에 거기서 설정을 토글/조절**한다. 산재한 체크박스를 폐지하고 한 줄에서 다 제어 = 일관성.

- **표기 관례(정본)**:
  - 라벨(헤더/오버레이/축약) = **mut 색**(`.gs-lbl`)
  - 활성 값 = **accent 네온그린**(`.gs-v.on`)
  - **옵션 구분 = `|`**(OPA\|합성본) · **그룹 구분 = 공백**(` / ` 안 씀) · **`[ ]` 대괄호 안 씀**
  - **배경·축약 = OFF/ON 단일 토글**(`syncOnOff`·`.gs-tog` 탭) — *흰칸은 고정*, 배경(흰칸 없는 nobg) 추가 여부만(기본 OFF·ON이면 +1장). "기본\|흰"·"[흰칸\|배경]" 식 2값 나열 폐지 = 혼란 → 단일 토글 통일(260624). OPA = `.gs-opa`(휠·상하드래그·조절중 mut→고정 accent)
  - 기본(combo) goSpec = **90%(11.25px) + nowrap** = 모바일 한 줄(`#goSpec`)
- **적용 범위**: 릴스 3모드(`#goSpec` combo·`#hdrSpec` header·`#ovSpec` overlay) + 포스트(`#postSpec` 축약). **저작권·카드뉴스는 입력 UI 본체라 goSpec 제외.**
- **구현(흰=고정 정신)**: hidden input(`#cBg`·`#rBg`·`#abbr`) 유지 + goSpec span이 그 상태를 토글 → **발사(buildParams·go.onclick) 로직 불변.** `updateGoSpec()`가 모드별 명세를 갱신.

## §4. 입력 위젯 패턴 (기틀)

- **텍스트 + 이미지 = covrow**: 텍스트(2/3 `.covta`) + 이미지 슬롯(1/3 `.covimg`). 적용 = 기본 자막·오버레이·**포스트 배경**(260624). *full-width 이미지 행 금지(한 행 낭비) — 슬롯으로 통일.* ⚠️ 클립(복사/붙여넣기) 버튼과 호환 = `attachCopyPaste`가 `covta`를 wrap에 승계(안 하면 wrap이 flex 먹어 슬롯 찌부 = 260624 버그).
  - 바인더 = **`bindSlotImg(slotSel, inputSel, store)`** — 기본(`#cImgSlot`/`CIMG`)·오버레이(`#rImgSlot`/`RIMG`)·**포스트(`#pImgSlot`/`PIMG` = 카드뉴스 `IMG`와 분리 = 위젯 desync 방지)** 공용. 슬롯 탭=선택 · X=삭제 · `_set(b64,name)`/`_reset()`=수정복원용.
- **단독 이미지(배경) = `bindFile(input, lab, store)`** — 포스트·카드뉴스 배경(`#file`/`#fileLab`/`IMG`). `_set`/`_reset` 동일 노출.
- **부제/제목** = `.cf-sub`(흰 입력)·`.cf-title`(초록 입력) = **결과물 색 미러**. 라벨 없이 placeholder(부제/제목). 기본·헤더 동일.
- **rfab(빠른 메뉴)** 이미지 버튼 = `imgInput()`이 현재 모드 file input(`#file`/`#rFile`/`#cFile`) 반환 → `.click()`.

## §5. OPA·축약·자간 관례

- **OPA**:
  - 포스트(`#opaSeg`): **58 기본** · 30 · 직접입력. 다중선택 = 장수.
  - 오버레이(`#rOpaSeg`): **60 기본(단일)** · 30 · 직접입력(260624 — 전 60+30 둘 다 → 60만).
  - 기본(combo): `COMBO_OPA` 단일값(goSpec OPA 휠·60 기본).
  - 최소 1개 유지(마지막 1개 못 끔 = 0 방지). 직접입력 = 값 있을 때만.
- **축약(abbr)**: goSpec **ON 토글**(전 탭 통일 · 버튼 안 체크박스 폐지). 화이트리스트 4종 = `...→⋯ · •→· · 이,→李, · 윤,→尹,`. **포스트·릴스만**(저작권·경고문·카드 제외).
- **자간(tracking) 안내**: 텍스트가 좌우 마진 초과하면 안내(`.trk`) — **노랑(`.on`)=자간 좁힘 · 빨강(`.over`)=축약 필요**. *초과할 때만* 표시.
  - `updateTrkHint(taId, hintId, fmt)` = 포스트(post)·오버레이(reels) 자막.
  - `renderHdrTrk(elSel, subSel, titleSel)` = 부제/제목(reels2 `_fit_tr`).
  - **전 탭 동일 적용**(포스트·헤더·오버레이·기본). 기본 = `updateComboTrk()`(헤더+자막 둘 다). 입력 시 실시간.

## §6. 수정(편집) = 제작 조건 복원 (기틀 · 260624)

> **목표**: 이전 제작을 작성기로 그대로 되돌려 **그 건에 맞는 탭으로 복귀** + 입력 복원.

- 발사 시 **`snapForm()`** 이 폼 설정(탭·모드·포맷·텍스트·OPA·토글·프리셋·카드칩)을 압축 스냅샷 → `dispatchJob`의 `job.src` → 이력(`histAdd`)·최근결과(`saveLastResult`)에 동봉.
- 이전 제작 카드 **다운로드 옆 연필(`.imgedit`)** → **`importToEditor(src)`**: `selApp`/`selRmode`로 탭 전환 + 입력 전부 복원 + 맨 위 스크롤.
- **텍스트·설정 = localStorage**(가벼움) · **이미지 b64 = IndexedDB**(`nomute_thumb_img` · 같은 기기 대용량 · 12h prune). 키 없음/만료/**타 기기 이력 = 이미지 재선택 안내**(연필은 로컬 `src` 있는 항목만 표시).
- 콤보는 1 이미지키 공유(헤더+자막 동일).

## §8. 기틀 vs 주변부 (흔들림 방지 — 제일 중요)

- **기틀(바꾸려면 §기틀 보호 = 사용자 확인 먼저)**: 탭 구조·디스패치 경로 · **흰=고정/백엔드 불변** · goSpec 패턴(표기 관례 포함) · covrow/`bindSlotImg` 패턴 · 수정복원(`snapForm`/IDB) · 자간 안내 체계 · 디자인 토큰 계승 · LLM 0.
- **주변부(자유 — 말 안 해도 됨)**: 문구·placeholder·기본값 미세조정(OPA·여백·색 미세) · 라벨 텍스트 · 안내문. **단 이걸 바꿔도 위 기틀 패턴은 따른다**(새 다운로드 버튼=`.dlbtn`, 새 이미지칸=covrow+`bindSlotImg`, 새 설정=goSpec 토글 등).
